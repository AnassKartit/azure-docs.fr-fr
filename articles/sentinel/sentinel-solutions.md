# <a name="about-azure-sentinel-solutions"></a>À propos des solutions Azure Sentinel

> [!IMPORTANT]
>
> L’expérience des solutions Azure Sentinel est actuellement en **préversion**, tout comme les packages de solutions individuels. Consultez l’[Avenant aux conditions d’utilisation pour les préversions de Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/) pour connaître les conditions juridiques supplémentaires s’appliquant aux fonctionnalités Azure sont en version bêta, en préversion ou non encore en disponibilité générale.

Les solutions Azure Sentinel assurent la détectabilité des produits, le déploiement en une seule étape et l’activation de scénarios de produits, de domaines et/ou verticaux de bout en bout dans Azure Sentinel. Cette expérience est basée sur la [Place de marché Azure](https://azuremarketplace.microsoft.com/marketplace) pour la détectabilité, le déploiement et l’activation des solutions, ainsi que sur [l’Espace partenaires Microsoft](/partner-center/overview) pour la création et la publication de solutions.

## <a name="why-azure-sentinel-solutions"></a>Pourquoi choisir les solutions Azure Sentinel ?

- Les clients peuvent facilement découvrir le contenu empaqueté et les intégrations qui apportent de la valeur aux produits, domaines ou verticaux dans Azure Sentinel.

- Les clients peuvent facilement déployer le contenu en une seule étape et activer le contenu pour commencer immédiatement s’ils le souhaitent.

- Les fournisseurs ou partenaires peuvent offrir des solutions combinées d’ajout de valeur pour les produits, domaines ou verticaux dans Azure Sentinel, permettant ainsi la productisation des investissements.

## <a name="types-of-azure-sentinel-solutions"></a>Types de solutions Azure Sentinel

Azure Sentinel offre actuellement des solutions de **contenu empaqueté**. [Ces solutions](sentinel-solutions-catalog.md) incluent des combinaisons d’un ou plusieurs connecteurs de données, classeurs, règles d’analyse, playbooks, requêtes de chasse, analyseurs, watchlists et d’autres composants pour Azure Sentinel. Découvrez comment [rechercher et déployer des solutions de contenu empaqueté](sentinel-solutions-deploy.md).

Deux autres types de solutions pour Azure Sentinel peuvent être proposés à l’heure actuelle dans la Place de marché Azure générique, bien qu’ils n’apparaissent pas dans la Galerie Solutions Azure Sentinel :

- **Intégrations** : comprend des services ou des outils créés à l’aide des API Azure Sentinel ou des API Azure Log Analytics pour permettre aux clients d’intégrer leurs applications existantes à Azure Sentinel ou de migrer des données, des requêtes, etc. à partir d’applications existantes vers Azure Sentinel.

- **Offres de service** : comprend les listes de services managés spécifiques pour Azure Sentinel.
