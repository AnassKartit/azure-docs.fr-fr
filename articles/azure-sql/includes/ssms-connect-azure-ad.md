> [!NOTE]
> En décembre 2021, les versions de SSMS antérieures à la version 18.6 ne s’authentifieront plus via Azure Active Directory avec l’authentification multifacteur.
>
> Pour continuer à utiliser l’authentification Azure Active Directory avec MFA, vous devez avoir [SSMS 18.6 ou ultérieur](/sql/ssms/download-sql-server-management-studio-ssms).
