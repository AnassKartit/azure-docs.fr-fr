# <a name="azure-policy-built-in-definitions-for-azure-hdinsight"></a>Définitions intégrées d’Azure Policy pour Azure HDInsight

Cette page représente un index des définitions de stratégie intégrées [Azure Policy](../governance/policy/overview.md) pour Azure HDInsight. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../governance/policy/samples/built-in-policies.md).

Le nom de chaque définition de stratégie intégrée est un lien vers la définition de la stratégie dans le portail Azure. Utilisez le lien de la colonne **Version** pour voir la source dans le [dépôt GitHub Azure Policy](https://github.com/Azure/azure-policy).
