Dans la boîte de dialogue **Nouvelle table**, remplissez le formulaire comme ci-dessous.

1. Entrez le nom **WeatherData** comme ID de table.  Il s’agit du nom de la table.
1. Sélectionnez **Manuel** sous *Débit de la table (mise à l’échelle automatique)* pour cet exemple.
1. Utilisez la valeur par défaut 400 pour les unités de requête par seconde (RU/s).
1. Sélectionnez le bouton **OK** pour créer la table.
