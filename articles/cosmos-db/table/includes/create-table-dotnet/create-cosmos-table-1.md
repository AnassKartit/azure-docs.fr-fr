Dans le portail Azure, accédez à la page Vue d’ensemble du compte Azure Cosmos DB.  Vous pouvez accéder à la page Vue d’ensemble de votre compte Cosmos DB en tapant son nom (cosmos-msdocs-tables-sdk-demo-XYZ) dans la barre de recherche supérieure et en regardant sous l’en-tête des ressources.

Sélectionnez le nom de votre compte Azure Cosmos DB pour accéder à la page Vue d’ensemble correspondante.
