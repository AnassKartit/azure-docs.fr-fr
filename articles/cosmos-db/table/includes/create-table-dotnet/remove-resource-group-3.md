Une boîte de dialogue s’affiche à droite de l’écran vous demandant de confirmer la suppression du groupe de ressources.

1. Tapez le nom complet du groupe de ressources dans la zone de texte pour confirmer la suppression.
1. Sélectionnez le bouton *Supprimer* au bas de la page.
