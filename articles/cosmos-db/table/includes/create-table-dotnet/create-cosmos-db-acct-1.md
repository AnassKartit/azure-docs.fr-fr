Dans le portail Azure :

   1. Dans la barre de recherche située en haut du portail Azure, entrez « cosmos db ».
   1. Dans le menu qui s’affiche sous la barre de recherche, sous **Services**, sélectionnez l’élément intitulé **Azure Cosmos DB**.
