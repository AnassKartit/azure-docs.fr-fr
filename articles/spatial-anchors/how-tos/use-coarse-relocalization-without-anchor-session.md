# <a name="use-coarse-relocalization-without-an-anchor-session"></a>Utiliser la relocalisation approximative sans session d’ancre

L’objet fournisseur de capteur d’empreintes digitales inclus dans la fonctionnalité de [relocalisation grossière](../concepts/coarse-reloc.md) peut être utilisé sans être attaché à une session Azure Spatial Anchors.
L’extrait de code suivant montre comment le fournisseur de capteur est accessible pour obtenir une estimation de la position de l’appareil :

# <a name="c"></a>[C#](#tab/csharp)

```csharp
// Game about to start, start tracking the sensors
sensorProvider.Start();

// Game loop
while (m_isRunning)
{
    // Get the GPS estimate
    GeoLocation geoPose = sensorProvider.GetLocationEstimate();

    // Paint it on the map
    drawCircle(
        x: geoPose.Longitude,
        y: geoPose.Latitude,
        radius: geoPose.HorizontalError);
}

// Game ended, no need to track the sensors anymore
sensorProvider.Stop();
```
