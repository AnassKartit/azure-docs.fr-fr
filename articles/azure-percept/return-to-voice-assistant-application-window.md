# <a name="find-your-voice-assistant-application-in-azure-percept-studio"></a>Trouver l’application d’Assistant vocal dans Azure Percept Studio

Ce guide pratique indique comment revenir à une application d’assistant vocal créée précédemment.

## <a name="prerequisites"></a>Prérequis

- [Créer une application de démonstration d’assistant vocal](./tutorial-no-code-speech.md)
- Votre Azure Percept DK est alimenté et l’accessoire Azure Percept Audio est connecté à l’aide d’un câble USB.

## <a name="open-your-voice-assistant-application"></a>Ouvrir votre application d’assistant vocal
1. Accédez à [Azure Percept Studio](https://portal.azure.com/#blade/AzureEdgeDevices/Main/overview).
1. Dans le volet du menu de gauche, sélectionnez **Appareils**.
    :::image type="content" source="media/return-to-voice-assistant-demo-window/select-device.png" alt-text="Capture d’écran montrant la sélection de la section Appareil dans le volet du menu de gauche":::
1. Sélectionnez l’appareil sur lequel votre application Assistant vocal a été déployée.
1. Sélectionnez l’onglet **Speech**.  :::image type="content" source="media/return-to-voice-assistant-demo-window/speech-tab.png" alt-text="sélectionnez l’onglet Speech":::
1. Sous **Actions**, sélectionnez **Tester votre Assistant vocal** :::image type="content" source="media/return-to-voice-assistant-demo-window/actions-test-va.png" alt-text="sélectionnez votre test d’assistant vocal sous la section Actions":::

## <a name="next-steps"></a>Étapes suivantes
Maintenant que votre application de l’Assistant vocal est ouverte, essayez d’effectuer d’[autres configurations](./how-to-manage-voice-assistant.md).
