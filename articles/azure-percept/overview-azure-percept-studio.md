# <a name="azure-percept-studio-overview"></a>Vue d’ensemble d’Azure Percept Studio

Azure Percept Studio est l’unique point de départ pour la création de modèles et de solutions d’IA Edge. Azure Percept Studio vous permet de découvrir et d’effectuer des workflows guidés qui facilitent l’intégration de matériel compatible avec l’intelligence artificielle Edge, ainsi que l’intégration de services cloud Azure AI et IoT Cloud puissants.

## <a name="features"></a>Fonctionnalités

Dans le Studio, vous verrez vos appareils d’intelligence artificielle Edge comme des points de terminaison pour la collecte des données initiales et des données d’entraînement continu, ainsi que comme des cibles de déploiement pour les itérations de modèle. Le fait d’avoir accès aux appareils et aux données d’entraînement permet un prototypage rapide et un développement de modèles d’IA Edge itératif pour les scénarios de [vision](./tutorial-nocode-vision.md) et de [reconnaissance vocale](./tutorial-no-code-speech.md).

![Flux du projet à partir de la page d’accueil dans Azure Percept Studio.](./media/overview-azure-percept-studio/percept-studio-flow.png)

Dans Azure Percept Studio, les workflows intègrent de nombreux services Azure AI et IoT sous-jacents, comme Azure IoT Hub, Custom Vision, Speech Studio et Azure ML Services, ce qui vous permet d’utiliser ces services pour créer une solution de bout en bout, sans avoir de connaissances préalables importantes. Si vous êtes déjà familiarisé avec ces services Azure, vous pouvez également vous connecter à des ressources existantes et les modifier en dehors d’Azure Percept Studio.

![Flux des appareils dans Azure Percept Studio.](./media/overview-azure-percept-studio/device-flow.png)

Que vous soyez un développeur débutant ou un développeur de solutions et de modèles d’IA expérimenté, que vous travailliez à un prototype ou que vous passiez à une solution de production, et que votre projet d’IA Edge concerne la reconnaissance vocale ou la vision, le portail Azure Percept Studio offre un accès à des workflows qui facilitent la création de solutions d’intelligence artificielle Edge.

![Flux de capture d’image dans Azure Percept Studio.](./media/overview-azure-percept-studio/image-flow.png)
