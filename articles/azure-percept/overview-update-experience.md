# <a name="the-azure-percept-dk-update-experience"></a>L’expérience de mise à jour du DK Azure Percept

Avec le DK Azure Percept, vous disposez de deux options pour mettre à jour le système d’exploitation et le microprogramme de votre devkit : OTA (over-the-air) ou via USB. La mise à jour OTA permet de maintenir facilement à jour les appareils. La mise à jour USB, elle, est une bonne option quand la mise à jour OTA n’est pas possible ou quand vous voulez réinitialiser votre appareil aux paramètres d’usine. Pour vous permettre de tirer parti de la méthode de mise à jour qui vous convient, nous avons rassemblé une collection de guides pratiques pour vous aider. 

- [Guide pratique pour configurer Azure IoT Hub pour déployer des mises à jour OTA sur votre DK Azure Percept](./how-to-set-up-over-the-air-updates.md)
- [Guide pratique pour mettre à jour votre DK Azure Percept via OTA](./how-to-update-over-the-air.md)
- [Guide pratique pour mettre à jour votre DK Azure Percept via USB](./how-to-update-via-usb.md)
