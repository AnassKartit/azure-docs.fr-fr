# <a name="migrate-to-azure-app-service"></a>Migrer vers Azure App Service

À l’aide d’[App Service Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/), vous pouvez migrer votre application locale sur Azure App Service. App Service Migration Assistant est destiné à simplifier votre parcours vers le cloud grâce à une solution gratuite, simple et rapide conçue pour migrer des applications locales vers le cloud.

Grâce à Azure App Service Migration Assistant, vous pouvez rapidement :

- analyser l’URL de votre application pour déterminer si elle est un bon candidat à la migration ;
- télécharger l’Assistant Migration pour commencer la migration ;
- utiliser l’outil pour exécuter des vérifications de préparation et une évaluation générale des paramètres de configuration de votre application ;
- migrer votre application ou votre site vers Azure App Service via l’outil.

[Regardez comment migrer des applications web vers Azure App Service](https://www.youtube.com/watch?v=9LBUmkUhmXU).

Étape suivante : [Migrer une application web locale vers Azure App Service](https://docs.microsoft.com/learn/modules/migrate-app-service-migration-assistant/)
