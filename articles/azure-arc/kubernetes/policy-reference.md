# <a name="azure-policy-built-in-definitions-for-azure-arc-enabled-kubernetes"></a>Définitions intégrées Azure Policy pour Kubernetes avec Azure Arc

Cette page est un index des définitions de stratégie intégrées [Azure Policy](../../governance/policy/overview.md) pour Kubernetes avec Azure Arc. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../../governance/policy/samples/built-in-policies.md).
