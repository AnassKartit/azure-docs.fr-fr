# <a name="azure-arc-enabled-sql-managed-instance-overview"></a>Vue d’ensemble de SQL Managed Instance avec Azure Arc

SQL Managed Instance avec Azure Arc est un service de données Azure SQL qui peut être créé sur l’infrastructure de votre choix. 


## <a name="description"></a>Description

SQL Managed Instance avec Azure Arc présente une compatibilité proche de 100 % avec le moteur de base de données SQL Server le plus récent et permet aux clients actuels de SQL Server de lift-and-shift leurs applications vers les services de données Azure Arc sans changements importants des applications et des bases de données et tout en préservant la souveraineté des données. En outre, SQL Managed Instance intègre des fonctionnalités de gestion intégrées qui réduisent considérablement la surcharge de gestion.
