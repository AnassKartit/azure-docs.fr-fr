# <a name="product-updates-for-postgresql---hyperscale-citus"></a>Mises à jour produit pour PostgreSQL - Hyperscale (Citus)

## <a name="updates-feed"></a>Flux de mises à jour

Le site web Microsoft Azure liste les fonctionnalités nouvellement disponibles pour chaque produit, ainsi que les fonctionnalités en préversion et en cours de développement. Consultez la section [Mises à jour Hyperscale (Citus)](https://azure.microsoft.com/updates/?category=databases&query=citus) pour obtenir la dernière version des fonctionnalités. Un flux RSS est également disponible dans cette page.

## <a name="features-in-preview"></a>Fonctionnalités de la version préliminaire

Azure Database pour PostgreSQL - Hyperscale (Citus) propose des préversions pour les fonctionnalités non publiées. Les préversions sont fournies sans contrat de niveau de service et sont déconseillées pour les charges de travail de production. Certaines fonctionnalités peuvent être limitées ou non prises en charge.  Pour plus d’informations, consultez [Conditions d’utilisation supplémentaires relatives aux préversions de Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).

Voici les fonctionnalités actuellement disponibles pour la préversion :

* **[pgAudit](concepts-hyperscale-audit.md)** . Fournit une journalisation détaillée des audits de session et d’objets via la fonctionnalité de journalisation PostgreSQL standard. Elle génère les journaux d’audit nécessaires pour passer certains audits administratifs, financiers ou de certification ISO.
* **[Accès privé](concepts-hyperscale-private-access.md)**.
  Autorisez les hôtes sur un réseau virtuel (VNet) à accéder en toute sécurité à un groupe de serveurs Hyperscale (Citus) sur un point de terminaison privé.

> [!NOTE]
>
> L’accès privé est disponible en préversion dans [certaines régions](concepts-hyperscale-limits.md#regions) uniquement.
