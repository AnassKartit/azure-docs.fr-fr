# <a name="azure-storage-analytics-and-big-data-partners"></a>Partenaires de Big Data et d’analytique avec le stockage Azure

Cet article présente les sociétés partenaires de Microsoft qui sont intégrées à Azure Data Lake Storage. Ces solutions de partenaires couvrent les charges de travail telles que les entrepôts de données modernes, l’analytique avancée et l’analytique en temps réel. Ces partenaires tirent parti de l’[espace de noms hiérarchique](../../../blobs/data-lake-storage-namespace.md) dans le stockage Azure pour optimiser leur solution et l’exécuter efficacement dans Azure.
