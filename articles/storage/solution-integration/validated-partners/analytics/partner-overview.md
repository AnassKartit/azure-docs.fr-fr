# <a name="azure-storage-analytics-and-big-data-partners"></a>Partenaires Azure Storage Analytics et Big Data

Cet article présente les sociétés partenaires de Microsoft qui sont intégrées à Azure Data Lake Storage. Elles couvrent les charges de travail, telles que les entrepôts de données modernes, l’analytique avancée, l’analytique en temps réel, etc. Ces partenaires tirent parti de la fonctionnalité Espace de noms hiérarchique dans Stockage Azure pour optimiser leur solution et l’exécuter efficacement dans Azure.

## <a name="verified-analytics-and-big-data-partners"></a>Partenaires Analytics et Big Data vérifiés

| ------- | ----------- | -------------------- |

Dremio facilite la jonction d’ADLS avec Stockage Blob, Azure SQL Database, Azure Synapse SQL, HDInsight et bien plus encore. Avec Dremio, les analystes de Power BI peuvent rechercher de nouveaux jeux de données stockés sur ADLS, accéder immédiatement à ces données dans Power BI sans aucune préparation effectuée par le service informatique, créer des visualisations et affiner les rapports en temps réel de manière itérative.
