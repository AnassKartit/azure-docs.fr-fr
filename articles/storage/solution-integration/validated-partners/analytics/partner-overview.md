# <a name="azure-storage-analytics-and-big-data-partners"></a>Partenaires de Big Data et d’analytique avec le stockage Azure

Cet article présente les sociétés partenaires de Microsoft qui sont intégrées à Azure Data Lake Storage. Ces solutions de partenaires couvrent les charges de travail telles que les entrepôts de données modernes, l’analytique avancée et l’analytique en temps réel. Ces partenaires tirent parti de l’[espace de noms hiérarchique](../../../blobs/data-lake-storage-namespace.md) dans le stockage Azure pour optimiser leur solution et l’exécuter efficacement dans Azure.

## <a name="verified-partners"></a>Partenaires vérifiés

| Partenaire | Description | Site web/lien produit |
| ------- | ----------- | -------------------- |

Dremio facilite la jonction de Data Lake Storage avec le stockage Blob, Azure SQL Database, Azure Synapse SQL, HDInsight, etc. Avec Dremio, les analystes de Power BI peuvent rechercher de nouveaux jeux de données stockés sur Data Lake Storage, accéder immédiatement à ces données dans Power BI, créer des visualisations et affiner les rapports en temps réel de manière itérative. Les analystes peuvent également créer des rapports qui combinent des données entre Data Lake Storage et d’autres bases de données.|[Page du partenaire](https://www.dremio.com/azure/)<br>[Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/dremiocorporation.dremio_ce)<br>|
![Logo de la société Informatica](./media/informatica-logo.png) |**Informatica**<br>La plateforme de gestion des données native cloud à l’échelle de l’entreprise d’Informatica automatise et accélère la découverte, la distribution, la qualité et la gouvernance des données d’entreprise sur Azure. Ses fonctionnalités d’intégration de données, de qualité des données et de gouvernance, pilotées par les métadonnées et basées sur l’intelligence artificielle, vous permettent de moderniser l’analytique et d’accélérer votre migration vers un entrepôt de données ou un lac de données sur Azure.|[Page du partenaire](https://www.informatica.com/azure)<br>[Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps/informatica.annualiics?tab=Overview)|
