# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Partenaires d’archivage, de sauvegarde et de récupération d’urgence du Stockage Azure

Cet article met en évidence les partenaires Microsoft offrant une intégration avec le Stockage Azure pour les charges de travail d’archivage, de continuité de l’activité et de récupération d’urgence. Ils tirent parti des avantages des avantages du Stockage Azure en termes d’échelle et de [coûts](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) pour résoudre efficacement les problèmes de sauvegarde, créer un site de récupération d’urgence et archiver les contenus inutilisés pour la rétention à long terme. Avec toutes les normes de conformité auxquelles le Stockage Azure satisfait, ainsi que le [stockage immuable](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) et la gestion du cycle de vie, ces solutions peuvent facilement remplacer les sauvegardes sur bande et proposer un site de récupération économique à la demande.
