# <a name="legal-terms-and-third-party-notices"></a>Termes juridiques et avis aux tiers
Cet article fournit des informations sur les termes juridiques et mentions relatives aux tiers associés à Event Grid sur Kubernetes. 

## <a name="legal-terms"></a>Termes et conditions 

Les termes juridiques régissant l’utilisation d’Event Grid sur Kubernetes avec Azure Arc figurent dans le système de fichiers de l’opérateur Event Grid et dans les conteneurs de l’agent Event Grid dans les répertoires suivants : 

Opérateur Event Grid : ``/LICENSE``.

Agent Event Grid : ``/app/LICENSE``.


## <a name="third-party-notices"></a>Mentions tierces

Le document présentant les mentions relatives aux logiciels tiers pour Event Grid sur Kubernetes avec Azure Arc figure dans le système de fichiers de l’opérateur Event Grid et dans les conteneurs de l’agent Event Grid dans les répertoires suivants :

Opérateur Event Grid : ``/NOTICE.txt``.

Agent Event Grid : ``/app/NOTICE.txt``.
