# <a name="how-to-find-your-azure-tenant-id"></a>Rechercher votre ID de locataire Azure

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Rechercher votre ID de locataire dans le portail

1. Connectez-vous au [portail Azure](https://portal.azure.com).
1. Sélectionnez l’icône Azure Active Directory sous Services Azure. La page Active Directory s’ouvre.
1. Votre ID de locataire se trouve sous **Informations de base**.
1. Copiez l’ID de locataire et collez-le dans un document texte de votre choix pour une utilisation ultérieure.
