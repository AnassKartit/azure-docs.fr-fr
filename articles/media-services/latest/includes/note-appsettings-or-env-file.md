Ouvrez appsettings.json dans votre projet téléchargé. Remplacez les valeurs par les informations d’identification que vous avez obtenues en [accédant aux API](../access-api-howto.md).

> [!NOTE]
> Vous pouvez également utiliser le format de fichier *.env* à la racine du projet afin de définir vos variables d’environnement une seule fois pour tous les projets du dépôt d’exemples .NET. Il vous suffit de copier le fichier *sample.env*, puis de renseigner les informations que vous avez obtenues à partir de la page **Accéder à l’API** de Media Services dans le portail Azure ou à partir d’Azure CLI. Renommez le fichier *sample.env* en *.env* pour l’utiliser dans tous les projets.<br/><br/>
> Le fichier *.gitignore* est déjà configuré pour empêcher la publication de ce fichier dans votre dépôt dupliqué.
