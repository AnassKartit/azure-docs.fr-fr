> [!WARNING]
> Il n’est pas recommandé de tenter d’envelopper l’API REST pour Media Services directement dans votre propre code de bibliothèque, car réaliser correctement cette opération à des fins de production vous oblige à implémenter la logique de nouvelle tentative complète de gestion des ressources Azure et à comprendre comment gérer les opérations durables dans les API de gestion des ressources Azure. Cette opération est gérée automatiquement pour vous par les SDK clients pour différents langages (.NET, Java, TypeScript, Python, Ruby, etc.) et réduit les risques de problèmes liés à la logique de nouvelle tentative ou d’échecs des appels d’API. Les kits SDK client gèrent déjà cette fonction pour vous. La collection Postman est fournie avant tout comme un outil d’enseignement et vous montre ce que les kits SDK client effectuent pendant votre développement avec les différents kits SDK clients.
