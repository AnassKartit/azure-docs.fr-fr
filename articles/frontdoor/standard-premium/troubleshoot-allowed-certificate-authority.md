# <a name="allowed-certificate-authorities-for-azure-front-door-standardpremium-preview"></a>Autorités de certification autorisées pour Azure Front Door Standard/Premium (préversion)

Lorsque vous activez la fonctionnalité HTTPS à l’aide de votre propre certificat pour un domaine personnalisé Azure Front Door Standard/Premium. Vous avez besoin d’une autorité de certification autorisée pour créer votre certificat TLS/SSL. Si vous utilisez une autorité de certification non autorisée ou un certificat auto-signé, votre demande sera rejetée.
