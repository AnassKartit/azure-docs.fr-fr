# <a name="whats-new-in-azure-synapse-analytics"></a>Quelles sont les nouveautés d’Azure Synapse Analytics ?

Cet article répertorie les mises à jour d’Azure Synapse Analytics disponibles depuis le mois d’octobre 2021.

Chaque mise à jour est liée au blog Azure Synapse Analytics et à un article fournissant des informations supplémentaires.

## <a name="october-2021-update"></a>Mise à jour d’octobre 2021

Les mises à jour suivantes sont nouvelles dans Azure Synapse Analytics ce mois-ci.

### <a name="general"></a>Général

- Gérer vos coûts avec les plans de préachat  Azure Synapse [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#manage-cost) [article](../cost-management-billing/reservations/synapse-analytics-pre-purchase-plan.md)
- Déplacer votre espace de travail Azure Synapse entre régions Azure [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#move-workspace-region) [article](how-to-move-workspace-from-one-region-to-another.md)

### <a name="apache-spark-for-synapse"></a>Apache Spark pour Synapse

- Optimisations des performances Spark [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#spark-performance) 

### <a name="security"></a>Sécurité

- Tous les rôles RBAC Synapse sont désormais généralement disponibles pour une utilisation en production [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#synapse-rbac) [article](./security/synapse-workspace-synapse-rbac-roles.md)
- Tirer parti des identités managées affectées par l’utilisateur pour le chiffrement double [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#user-assigned-managed-identities) [article](./security/workspaces-encryption.md)
- Les administrateurs Synapse disposent désormais d’un accès élevé aux pools SQL dédiés [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#elevated-access) [article](./security/synapse-workspace-access-control-overview.md)
    
###  <a name="governance"></a>Gouvernance

- Les espaces de travail Synapse peuvent désormais envoyer (push) automatiquement des données de traçabilité à Azure Purview [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#synapse-purview-lineage) [article](../purview/how-to-lineage-azure-synapse-analytics.md)
 
### <a name="integrate"></a>Integrate

- Utiliser Stringify dans les flux de données pour transformer facilement des types de données complexes en chaînes [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#stringify-transform) [article](../data-factory/data-flow-stringify.md)
- Contrôler la durée de vie de session Spark dans les flux de données [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#data-flowspark-ttl) [article](../data-factory/concepts-integration-runtime-performance.md)

### <a name="cicd--git"></a>CI/CD et Git

- Déployer des espaces de travail Synapse à l’aide de GitHub Actions [blog](https://techcommunity.microsoft.com/t5/azure-synapse-analytics/azure-synapse-analytics-october-update/ba-p/2875372#deploy-synapse-github-action) [article](./cicd/continuous-integration-delivery.md#configure-github-actions-secrets)
