1. Dans Synapse Studio, dans le volet de gauche, sélectionnez **Gérer** > **Pools Data Explorer**.
1. Sélectionnez le pool Data Explorer à utiliser pour voir ses détails.

    :::image type="content" source="../media/ingest-data-pipeline/select-data-explorer-pool.png" alt-text="Capture de l’écran des pools Data Explorer, montrant la liste des pools existants.":::

1. Notez les points de terminaison de requête et d’ingestion de données. Utilisez le point de terminaison de requête comme cluster pour la configuration des connexions à votre pool Data Explorer.

    :::image type="content" source="../media/ingest-data-pipeline/select-data-explorer-pool-properties-endpoints.png" alt-text="Capture d’écran du volet de propriétés des pools Data Explorer, montrant les adresses d’URI de l’ingestion de données et des requêtes.":::
