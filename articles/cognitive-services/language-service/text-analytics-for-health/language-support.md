# <a name="language-support-for-text-analytics-for-health"></a>Prise en charge des langages pour Analyse de texte pour le domaine de la santé

> [!NOTE]
> * Le conteneur utilise des versions de modèle différentes de celles des points de terminaison d’API et du kit de développement logiciel (SDK).
> * Les langages sont ajoutés à mesure que de nouvelles versions de modèle sont publiées. Les [versions de modèle](how-to/call-api.md#specify-the-text-analytics-for-health-model) actuelles pour l’Analyse de texte pour la santé sont :
>    * API et SDK : `2021-05-15`
>    * Conteneur : `2021-03-01`

Dans cet article, découvrez les langages naturels pris en charge par Analyse de texte pour le domaine de la santé et son conteneur Docker. 

## <a name="rest-api-and-client-library"></a>API REST et bibliothèque de client

| Langage | Code langue | À partir de la version de modèle v3 : | Notes |
|:---------|:-------------:|:-----------------------------------------:|:-----:|
| Anglais  |     `en`      |                Point de terminaison d’API et bibliothèque de client : 2019-10-01                 |       |

## <a name="docker-container"></a>Conteneur Docker

| Langage | Code langue | À partir de la version de modèle v3 : | Notes |
|:---------|:-------------:|:-----------------------------------------:|:-----:|
| Anglais  |     `en`      |                Conteneur : 2020-04-16                |       |
