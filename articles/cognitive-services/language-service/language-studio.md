# <a name="quickstart-get-started-with-language-studio"></a>Démarrage rapide : prise en main de Language Studio

[Language Studio](https://aka.ms/languageStudio) est un ensemble d’outils basés sur une interface utilisateur qui vous permet d’explorer, de créer et d’intégrer des fonctionnalités d’Azure Cognitive Service pour Language dans vos applications.

Language Studio vous offre une plateforme permettant d’essayer plusieurs fonctionnalités du service et de voir ce qu’elles retournent de manière visuelle. Il offre également une expérience facile à utiliser pour créer des projets et des modèles personnalisés pour travailler sur vos données. À l’aide de Studio, vous pouvez commencer sans avoir à écrire de code, puis utiliser les bibliothèques de client et les API REST disponibles dans votre application.

## <a name="language-studio-pre-configured-features"></a>Fonctionnalités préconfigurées de Language Studio

Le service Language offre plusieurs fonctionnalités qui utilisent des modèles prédéfinis préconfigurés pour effectuer diverses tâches, telles que la liaison d’entités, la détection de la langue et l’extraction de phrases clés. Pour obtenir la liste des fonctionnalités offertes par le service, consultez la [présentation d’Azure Cognitive Service pour Language](overview.md). 

Chacune de ces fonctionnalités offre une expérience de type démonstration dans Language Studio qui vous permet d’entrer du texte et présente la réponse à la fois visuellement et en JSON. Ces démonstrations vous aident à tester rapidement ces fonctionnalités préconfigurées sans utiliser de code.

## <a name="language-studio-customizable-features"></a>Fonctionnalités personnalisables Language Studio

Le service Language offre également plusieurs fonctionnalités qui vous permettent de créer, de former et de déployer des modèles personnalisés pour mieux les adapter à vos données. Par exemple, la classification de contenu personnalisée et la réponse à une question personnalisée. Pour les fonctionnalités de personnalisation, Language Studio offre des flux de travail qui permettent aux développeurs et aux experts de créer des modèles sans avoir besoin d’une expertise en Machine Learning. 

## <a name="clean-up-resources"></a>Nettoyer les ressources

Si vous souhaitez nettoyer et supprimer un abonnement Cognitive Services, vous pouvez supprimer la ressource ou le groupe de ressources. La suppression du groupe de ressources efface également les autres ressources qui y sont associées.

* [Portail](../cognitive-services-apis-create-account.md#clean-up-resources)
* [Azure CLI](../cognitive-services-apis-create-account-cli.md#clean-up-resources)

> [!TIP]
> Dans Language Studio, vous pouvez trouver les détails de votre ressource (par exemple, son nom et son niveau tarifaire), ainsi que les ressources en procédant comme suit :
> 1. En sélectionnant l’icône **Paramètres** dans le coin supérieur droit de l’écran Language Studio). 
> 2. Sélectionnez **Ressources**
>
> Vous ne pouvez pas supprimer votre ressource de Language Studio.
