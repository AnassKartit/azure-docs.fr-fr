# <a name="text-summarization-language-support"></a>Prise en charge du langage de synthèse de texte

Utilisez cet article pour découvrir les langages naturels pris en charge par la fonctionnalité de synthèse de texte. 

## <a name="languages-supported-by-text-summarization"></a>Langues prises en charge par la synthèse de texte 

La synthèse de texte prend en charge les langues suivantes :

| Langage              | Code langue |  À partir de la version de modèle v3 |     Notes          |
|:----------------------|:-------------:|:-------------------------------:|:------------------:|
| Chinois simplifié    | `zh-hans`     | 2021-08-01                      | `zh` également accepté |
| Anglais               | `en`          | 2021-08-01                      |                    |
| Français                | `fr`          | 2021-08-01                      |                    |
| Allemand                | `de`          | 2021-08-01                      |                    |
| Italien               | `it`          | 2021-08-01                      |                    |
| Japonais              | `ja`          | 2021-08-01                      |                    |
| Coréen                | `ko`          | 2021-08-01                      |                    |
| Espagnol               | `es`          | 2021-08-01                      |                    |
| Portugais (Brésil)   | `pt-BR`       | 2021-08-01                      |                    |
| Portugais (Portugal) | `pt-PT`       | 2021-08-01                      | `pt` également accepté |
