# <a name="personally-identifiable-information-pii-detection-language-support"></a>Prise en charge des langages par la fonctionnalité de détection des informations d’identification personnelle 

Utilisez cet article pour connaître les langages naturels pris en charge par la fonctionnalité de détection des informations d’identification personnelle d’Azure Cognitive Service for Language.

> [!NOTE]
> * Les langages sont ajoutés à mesure que de nouvelles [versions de modèle](how-to-call.md#specify-the-pii-detection-model) sont publiées.
> * La version de modèle actuelle pour les PII est `2021-01-15`.

## <a name="pii-language-support"></a>Prise en charge des langages pour la détection des informations d’identification personnelle

| Langage              | Code langue | À partir de la version de modèle v3 : | Notes              |
|:----------------------|:-------------:|:-------------------------------:|:------------------:|
| Chinois simplifié    | `zh-hans`     | 15-01-2021                      | `zh` également accepté |
| Anglais               | `en`          | 2020-07-01                      |                    |
| Français                | `fr`          | 15-01-2021                      |                    |
| Allemand                | `de`          | 15-01-2021                      |                    |
| Italien               | `it`          | 15-01-2021                      |                    |
| Japonais              | `ja`          | 15-01-2021                      |                    |
| Coréen                | `ko`          | 15-01-2021                      |                    |
| Portugais (Brésil)   | `pt-BR`       | 15-01-2021                      |                    |
| Portugais (Portugal) | `pt-PT`       | 15-01-2021                      | `pt` également accepté |
| Espagnol               | `es`          | 2020-04-01                      |                    |
