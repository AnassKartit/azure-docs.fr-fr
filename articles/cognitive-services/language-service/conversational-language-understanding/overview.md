# <a name="what-is-conversational-language-understanding-preview"></a>Qu’est-ce que le Language Understanding de conversation (version préliminaire) ?

Language Understanding de conversation est un service basé sur le Cloud qui vous permet de former des modèles de langage conversationnel à l’aide d’une technologie de pointe. 

L’API fait partie d’[Azure Cognitive Services](../../index.yml), une collection d’algorithmes de machine learning et d’intelligence artificielle dans le cloud pour vos projets de développement. Vous pouvez utiliser ces fonctionnalités avec l’API REST ou les bibliothèques client.

## <a name="features"></a>Fonctionnalités

Le Language Understanding de conversation applique une intelligence Machine Learning personnalisée au texte en langage naturel des conversations d’un utilisateur afin d’en prédire le sens général et d’en extraire des informations détaillées et pertinentes.

* Technologie avancée de compréhension du langage naturel utilisant des réseaux neuronaux avancés.
* Modèles de classification et d’extraction robustes et sémantiquement sensibles.
* Une expérience de création de modèle simplifiée à l’aide de Language Studio.
* Modèles multilingues en mode natif qui vous permettent d’effectuer un apprentissage dans un langage et de tester dans d’autres.
* Types de projets d’orchestration qui vous permettent de connecter des services, y compris d’autres projets de Language Understanding conversationnels, des bases de connaissances personnalisées de questions et des applications LUIS.
