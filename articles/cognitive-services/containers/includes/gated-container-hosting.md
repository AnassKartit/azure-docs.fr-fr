> [!NOTE]
> À compter du 22 septembre 2020, la plupart des conteneurs contrôlés sont hébergés sur Microsoft Container Registry. Il n’est par ailleurs pas nécessaire d’utiliser la commande docker login pour les télécharger. Il reste cependant indispensable d’effectuer une demande en ligne pour exécuter le conteneur. Pour plus d’informations, consultez la section **Demande d’approbation pour exécuter le conteneur** plus loin dans cet article.
