Quand vous utilisez la fonctionnalité Analyse de texte, les documents de votre requête ne doivent pas dépasser les limites de caractères suivantes. Pour plus d’informations, consultez l’[article sur les limites de données](../concepts/data-limits.md). 

* Opérations synchrones et Analyse de texte pour les opérations d’intégrité : 5 120 caractères par document.
* Opérations asynchrones (analyses) : 125 000 caractères par document.
