# <a name="troubleshooting-the-multivariate-api"></a>Résolution des problèmes liés à l’API multivariée

Cet article fournit des conseils sur la façon de dépanner et de corriger les messages d’erreur HTTP courants lors de l’utilisation de l’API multivariée.

### <a name="multivariate-error-codes"></a>Codes d’erreur Détecteur d’anomalies (multivarié)

| Méthode                                       | Code d'erreur HTTP | Message d’erreur                                                                                                            | Action à effectuer                                                                                                                                                                                                         |
|----------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The 'source' field is required in the request.`                                                                         | Le mot clé « source » n’a pas été spécifié correctement. Le format doit être `"{\"source\": \|" <SAS   URL>\"}"`                                                                                          |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The source field must be a valid sas blob url`                                                                          | Le champ source doit être une URL SAS de conteneur d’objets blob valide.                                                                                                                                                             |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The 'startTime' field is required in the request.`                                                                      | Ajoutez startTime à la requête                                                                                                                                                                                         |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The 'endTime' field is required in the request.`                                                                        | Ajoutez endTime à la requête                                                                                                                                                                                          |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `Invalid Timestamp format.`                                                                                              | Le timestamp du fichier CSV compressé dans l’URL source est dans un format non valide ou « startTime », « endTime » n’est pas dans un format valide.                                                                                 |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The displayName length exceeds maximum allowed length 24.`                                                              | displayName est un paramètre facultatif à utiliser pour permettre aux utilisateurs de distinguer les différents modèles. Un displayName valide doit comporter 24 caractères ou moins.                                                                    |
| Former un modèle Détecteur d’anomalies (multivarié) | 400             | `The 'slidingWindow' field must be an integer between 28 and 2880.`                                                      | La fenêtre glissante doit se trouver dans une plage valide.                                                                                                                                                                             |
| Former un modèle Détecteur d’anomalies (multivarié) | 401             | `Unable to download blobs on the Azure Blob storage account.`                                                            | L’URL ne dispose pas des autorisations appropriées. L’indicateur de liste n’est pas défini.   Le client doit recréer l’URL SAS et vérifier que les indicateurs de lecture et de liste sont activés (par exemple, à l’aide de l’Explorateur Stockage) |

Size of dataset exceeds size limit (2GB).`                                                   | Les données du conteneur d’objets blob dépassent la limite de 4 Mo actuellement. Le client doit pointer vers un objet blob avec des données plus petites.                                                                                          |
| Détecter les anomalies multivariées                  | 404             | `The model does not exist.`                                                                                              | L’ID de modèle n’est pas valide. | Détecter les anomalies multivariées                  | 400             | `The model is not ready yet.`                                                                                            | Le modèle n’est pas encore prêt.
