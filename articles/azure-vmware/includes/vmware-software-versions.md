Les versions du logiciel VMware utilisées dans les nouveaux déploiements de clusters de clouds privés Azure VMware Solution sont les suivantes :

| Logiciel              |    Version   |
| :---                  |     :---:    |
| VCSA / vSphere / ESXi |    6.7 U3l   | 
| ESXi                  |    6.7 U3l   | 
| vSAN                  |    6.7 U3l   |
| HCX                   |    4.1       |
| NSX-T <br />**Remarque :** NSX-T est la seule version prise en charge de NSX.               |      [3.1.2](https://docs.vmware.com/en/VMware-NSX-T-Data-Center/3.1/rn/VMware-NSX-T-Data-Center-312-Release-Notes.html)     |


La version actuelle du logiciel est appliquée aux nouveaux clusters ajoutés à un cloud privé existant. Pour plus d’informations, consultez la [configuration requise pour la version du logiciel VMware](https://docs.vmware.com/en/VMware-HCX/4.1/hcx-user-guide/GUID-54E5293B-8707-4D29-BFE8-EE63539CC49B.html).
