# <a name="security-solutions-for-azure-vmware-solution"></a>Solutions de sécurité pour Azure VMware Solution

La sécurité est un élément fondamental de Azure VMware Solution. Elle permet aux clients d’exécuter leurs charges de travail VMware dans un environnement sécurisé et fiable.

Nos partenaires de sécurité possèdent des solutions de pointe dans les environnements VMware qui couvrent de nombreux aspects de l’écosystème de sécurité tels que la protection contre les menaces et l’analyse de la sécurité. Nos clients ont adopté un grand nombre de ces solutions intégrées à VMware NSX-T pour leurs déploiements locaux. L’un de nos principes fondamentaux est de chercher à leur permettre de continuer à utiliser leurs investissements et leurs solutions VMware s’exécutant sur Azure. La plupart de ces éditeurs de logiciels indépendants ont validé leurs solutions avec Azure VMware Solution.

Vous pouvez obtenir plus d’informations au sujet de ces solutions ici :

- [Bitdefender](https://businessinsights.bitdefender.com/expanding-security-support-for-azure-vmware-solution)
- [Trend Micro Deep Security](https://www.trendmicro.com/en_us/business/products/hybrid-cloud/deep-security.html)
