Cette page constitue un index des définitions de stratégie intégrées [Azure Policy](../governance/policy/overview.md) pour Azure Data Box. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../governance/policy/samples/built-in-policies.md).
