# <a name="azure-policy-built-in-definitions-for-azure-attestation"></a>Définitions intégrées d’Azure Policy pour Azure Attestation

Cette page constitue un index des définitions de stratégie intégrées [d’Azure Policy](../governance/policy/overview.md) pour Azure Attestation. Pour obtenir des éléments intégrés supplémentaires d’Azure Policy pour d’autres services, consultez [Définitions intégrées d’Azure Policy](../governance/policy/samples/built-in-policies.md).
