# <a name="azure-policy-regulatory-compliance-controls-for-azure-logic-apps"></a>Contrôles de conformité réglementaire d’Azure Policy pour Azure Logic Apps

La [conformité réglementaire dans Azure Policy](../governance/policy/concepts/regulatory-compliance.md) fournit des définitions d’initiatives créées et gérées par Microsoft, qui sont dites _intégrées_, pour les **domaines de conformité** et les **contrôles de sécurité** associés à différents standards de conformité. Cette page liste les **domaines de conformité** et les **contrôles de sécurité** pour Azure Logic Apps. Vous pouvez affecter les initiatives intégrées pour un **contrôle de sécurité** individuellement, pour rendre vos ressources Azure conformes au standard spécifique.
