# <a name="get-support-for-datadog-resource"></a>Obtenir de l’aide pour une ressource Datadog

Cet article explique comment contacter l’assistance lorsque vous travaillez avec une ressource Datadog. Avant de contacter l’assistance technique, consultez [Corriger les erreurs courantes](troubleshoot.md).

## <a name="contact-support"></a>Contacter le support technique

Pour contacter le support technique de l’intégration de Datadog dans Azure, sélectionnez **Nouvelle demande de support** dans le volet gauche. Sélectionnez le lien vers le portail Datadog.

:::image type="content" source="media/get-support/support-request.png" alt-text="Créer une nouvelle demande de support" border="true":::

## <a name="next-steps"></a>Étapes suivantes

Pour obtenir des solutions potentielles, consultez [Corriger les erreurs courantes](troubleshoot.md).

Pour en savoir plus sur l’apport de modifications à votre ressource Datadog existante, consultez [Gérer la ressource Datadog](manage.md).
