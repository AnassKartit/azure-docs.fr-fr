# <a name="what-is-elastic-integration-with-azure"></a>Qu’est-ce que l’intégration d’Elastic à Azure ?

Cet article décrit l’application SaaS (Software as a Service) Elastic qui est disponible dans la Place de marché Azure. L’offre permet une intégration plus en profondeur du service Elastic à Azure.

La plateforme d’observabilité native Cloud d’Elastic centralise les journaux, les métriques et l’analyse du traçage dans un emplacement unique. Vous pouvez superviser facilement l’intégrité et les performances de votre environnement Azure. Ces informations vous aident à résoudre les problèmes de vos services plus rapidement.

Avec l’offre d’Elastic, vous pouvez gérer la solution sur le portail Azure. Vous implémentez Elastic comme solution de supervision pour vos charges de travail cloud au moyen d’un flux de travail rationalisé.

Vous pouvez maintenant provisionner les ressources Elastic à l’aide d’un fournisseur de ressources nommé **Microsoft.Elastic**. Vous créez, provisionnez et gérez les ressources Elastic sur le portail Azure. Elastic possède et exécute l’application SaaS, notamment les comptes Elastic créés.

## <a name="capabilities"></a>Fonctions

Voici les principales fonctionnalités fournies par l’intégration d’Elastic à Azure :

- **Intégration simplifiée** : utilisez le logiciel SaaS d’Elastic en tant que service intégré sur Azure.
- **Facturation unifiée** : gérez les coûts d’Elastic dans la facture mensuelle Azure. 
- **Authentification unique à Elastic** : aucune inscription distincte n’est nécessaire sur le portail Elastic.
- **Redirecteur de journal** : transfert des journaux d’activité d’abonnement et de ressources à Elastic.
- **Expérience de gestion** : emplacement unique pour la supervision des journaux d’expédition pour les services Azure envoyés à Elastic.
- **Déploiement de l’agent Elastic** : configurez des agents Elastic sur des hôtes de machines virtuelles en une seule étape.
