# <a name="what-is-logzio-integration-with-azure"></a>Qu’est-ce que l’intégration Logz.io à Azure ?

La plate-forme d’observabilité cloud native de Logz.io centralise les analyses de journal, de métrique et de suivi à un seul et même emplacement, ce qui vous permet de surveiller l’intégrité et les performances de votre environnement Azure. Vous gérez votre solution via le portail Azure.

L’offre Logz.io dans la place de marché Azure est un service intégré dans Azure. Du fait de cette intégration, vous pouvez implémenter Logz.io comme solution de surveillance pour vos charges de travail cloud au moyen d’un flux de travail rationalisé.

Les ressources Logz.io peuvent maintenant être provisionnées par le biais d’un fournisseur de ressources nommé `Microsoft.Logz`. Vous pouvez créer, provisionner et gérer les ressources Logz.io via le portail Azure. Logz.io possède et exécute l’application software as a service (SaaS) et qui comprend les comptes Logz.io et les clés API associées.

## <a name="capabilities"></a>Fonctionnalités

Les fonctionnalités clés suivantes sont fournies par l’intégration d’Azure Logz.io :

- **Intégration transparente** : le logiciel Saas Logz.IO est un service intégré sur Azure.
- **Facturation unifiée** : les coûts de Logz.io SaaS figurent sur une facture mensuelle Azure.
- **Authentification unique sur Logz.io.** -Aucune inscription distincte n’est nécessaire à partir du portail Logz.io.
- **Transfert de journaux** : active le transfert des journaux d’activité d’abonnement et de ressources vers Logz.io.
- **Expérience de gestion** : emplacement unique pour surveiller comment les journaux d’expédition des services Azure sont envoyés à Logz.io.
- **Déploiement de l’agent Logz.io** : configurer des agents Logz.io sur des hôtes de machines virtuelles en une seule étape.
