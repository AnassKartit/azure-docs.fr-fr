# <a name="view-or-change-your-settings-in-the-my-account-portal"></a>Afficher ou modifier vos paramètres dans le portail Mon compte

Vous pouvez consulter ou modifier les paramètres de votre compte, tels que la langue ou le fuseau horaire, via la page **Paramètres et confidentialité** du portail **Mon compte**.

>[!Note]
> Si vous voyez une erreur lors de la connexion avec un compte Microsoft personnel, vous pouvez toujours vous connecter en utilisant le nom de domaine de votre organisation (par exemple, contoso.com) ou l’**ID de locataire** de votre organisation obtenu auprès de votre administrateur dans l’une des URL suivantes :
>
>   - https://myaccount.microsoft.com?tenantId=*votre_nom_de_domaine*
>   - https://myaccount.microsoft.com?tenant=*votre_ID_de_locataire*

## <a name="view-and-manage-your-language-and-regional-settings"></a>Afficher et gérer vos paramètres linguistiques et régionaux

1. Connectez-vous à votre compte professionnel ou scolaire, puis accédez à votre page https://myaccount.microsoft.com/. Si vous vous connectez avec un compte Microsoft personnel, vous pouvez toujours vous connecter en utilisant le nom de domaine de votre organisation (par exemple, contoso.com) ou l’**ID de locataire** de votre organisation obtenu auprès de votre administrateur dans l’une des URL suivantes :

   - https://myaccount.microsoft.com?tenantId=*votre_nom_de_domaine*
   - https://myaccount.microsoft.com?tenant=*votre_ID_de_locataire*

1. Sélectionnez **Paramètres et confidentialité** dans le volet de navigation à gauche, ou sélectionnez le lien **Afficher les paramètres et la confidentialité** dans le bloc **Paramètres et confidentialité**.

    ![Page Mon compte, avec le lien Paramètres et confidentialité en surbrillance](media/my-account-portal/my-account-portal-privacy.png)

1. Sous *langue et région (préversion)* , consultez ou modifiez les paramètres suivants :
    * *Langue d’affichage* : langue utilisée pour les boutons, les menus et le contenu des sites web Microsoft.
    * *Langues par défaut* : langues que vous comprenez et utilisez pour créer et consommer du contenu.
    * *Format régional* : les sites web Microsoft utilisent ce paramètre pour mettre en forme les dates et les heures.
    * *Fuseau horaire* : utilisé pour planifier efficacement des réunions dans votre calendrier Microsoft.

Après avoir modifié la langue d’affichage, nous vous recommandons de vous déconnecter et de vous connecter à votre compte professionnel ou scolaire pour vous assurer que votre langue d’affichage est actualisée.

> [!NOTE]
> Dans certaines organisations, vous ne pouvez pas gérer le paramètre de *Langue d’affichage*. Si votre langue d’affichage est désactivée, comme dans l’exemple de capture d’écran suivant, contactez votre administrateur informatique pour obtenir de l’aide :
>
> ![Page Mon compte affichant les paramètres Langue d’affichage désactivés pour l’organisation](media/my-account-portal/my-account-portal-managed-language-settings.png)

## <a name="next-steps"></a>Étapes suivantes

Après avoir modifié les paramètres de votre portail Mon compte, vous pouvez :

- Afficher ou gérer vos [informations de sécurité](./security-info-setup-signin.md).

- Afficher ou gérer vos [appareils](my-account-portal-devices-page.md) connectés.

- Afficher ou gérer vos [organisations](my-account-portal-organizations-page.md).

- Afficher votre [activité de connexion](my-account-portal-sign-ins-page.md).

- Affichez les [applications web M365 qui utilisent actuellement les paramètres de format de langue et format régional](https://support.microsoft.com/office/change-your-display-language-and-time-zone-in-microsoft-365-for-business-6f238bff-5252-441e-b32b-655d5d85d15b).
