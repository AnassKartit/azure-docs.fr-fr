# <a name="tutorial-select-your-certification-program"></a>Tutoriel : Sélectionner votre programme de certification

Félicitations, vous avez choisi le programme Azure Certified Device ! Nous sommes heureux de vous voir rejoindre notre écosystème d’appareils certifiés. Pour commencer, vous devez déterminer les programmes de certification les mieux adaptés aux capacités de votre appareil.

Ce didacticiel vous apprend à effectuer les opérations suivantes :

> [!div class="checklist"]
> * Sélectionner le(s) meilleur(s) programme(s) de certification pour votre appareil

## <a name="selecting-a-certification-program-for-your-device"></a>Sélection d’un programme de certification pour votre appareil

Tous les appareils doivent répondre aux spécifications de référence décrites dans la certification [**Azure Certified Device**](./program-requirements-azure-certified-device.md). Les trois autres badges de certification s’appuient sur ce programme et fournissent une valeur client différente. Vous pouvez sélectionner un ou plusieurs des trois badges incrémentiels : IoT Plug-and-Play, Edge Managed et Edge Secured Core (préversion).

1. Consultez la configuration requise de chaque programme de certification dans le tableau ci-dessous. Vous pouvez sélectionner un en-tête de colonne pour découvrir les détails de la configuration requise de chaque programme concerné.

    |Condition requise|[IoT Plug and Play](./program-requirements-edge-secured-core.md)|[Edge Managed](./program-requirements-edge-managed.md)|[Edge Secured-core](./program-requirements-edge-secured-core.md)|
    ---|---|---|---
    | Processeur | Quelconque|MPU/Processeur|MPU/Processeur|
    | Système d’exploitation | Quelconque|[Système d’exploitation de niveau 1](../iot-edge/support.md?view=iotedge-2018-06&preserve-view=true))|[Système d’exploitation de niveau 1](../iot-edge/support.md?view=iotedge-2018-06&preserve-view=true)|
    | Runtime IoT Edge | Non pris en charge |Obligatoire|Obligatoire|
    | Defender pour IoT | Non pris en charge|Obligatoire|Obligatoire|
    | ADU/Windows Update | Non pris en charge|Obligatoire|Obligatoire|

1. Quand vous pensez avoir compris ce qu’il faut pour votre appareil, consultez les spécifications techniques du programme. Il peut s’agir de la certification Azure Certified Device, ou d’une combinaison de la certification de base et d’un des trois badges incrémentiels.
