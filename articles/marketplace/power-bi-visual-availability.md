# <a name="define-the-availability-of-a-power-bi-visuals-offer"></a>Définir la disponibilité d’une offre de visuel Power BI

Cette page vous permet de définir où et comment mettre votre offre à disposition (marchés et date de publication compris).

## <a name="markets"></a>Marchés

Pour spécifier les marchés sur lesquels votre offre doit être disponible, sélectionnez **Modifier les marchés**.

:::image type="content" source="media/power-bi-visual/markets.png" alt-text="Montre la page Disponibilité dans l’Espace partenaires, y compris la sélection des marchés.":::

Vos sélections s’appliquent uniquement aux nouvelles acquisitions. Si quelqu’un dispose déjà de votre application dans un marché et que vous supprimez ce marché, la personne pourra continuer à l’utiliser, mais aucun nouveau client de ce marché ne pourra bénéficier de votre offre.

>[!IMPORTANT]
>Il vous incombe de respecter les exigences légales locales, même si ces exigences ne sont pas répertoriées ici ou dans l’Espace partenaires. Même si vous sélectionnez tous les marchés, les lois et restrictions locales ou d’autres facteurs peuvent empêcher certaines offres d’être répertoriées dans certains pays et régions.

Si vous effectuez des modifications, sélectionnez **Enregistrer le brouillon** avant de passer à l'onglet suivant du menu de navigation gauche, **Configuration technique**.
