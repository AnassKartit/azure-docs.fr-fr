# <a name="set-the-preview-audience-for-an-iot-edge-module-offer"></a>Définir le public de préversion pour une offre Module IoT Edge

Sélectionnez **Public de préversion** pour choisir un public restreint pendant la préversion en vue de valider votre offre avant de la publier plus largement en ligne sur la Place de marché.

> [!IMPORTANT]
> Une fois que vous avez affiché votre offre en préversion, vous devez sélectionner **Mise en service** afin de publier votre offre pour le public.

## <a name="select-preview-audience-guids"></a>Sélectionner les GUID du public de préversion

Spécifiez votre public de préversion en utilisant au moins un des GUID d’ID d’abonnement Azure et une description facultative. Vous pouvez trouver votre ID d’abonnement Azure dans la page Abonnements du portail Azure. Aucun de ces champs ne peut être vu par les clients.

Ajoutez jusqu’à 10 GUID individuellement ou jusqu’à 100 en chargeant un fichier CSV. En ajoutant ces ID d’abonnement, vous définissez qui peut afficher un aperçu de votre offre avant qu’elle ne soit publiée en ligne. Si votre offre est déjà publiée, vous pouvez définir un public de préversion afin de tester des changements ou des mises à jour futurs.
