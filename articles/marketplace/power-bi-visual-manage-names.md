# <a name="manage-power-bi-visual-offer-names"></a>Gérer les noms des offres de visuels Power BI

Cette page vous permet de réserver des noms supplémentaires pour votre offre. Ces noms peuvent être utilisés pour une autre langue ou si vous décidez de changer le nom du produit.

Dans le menu de gauche, sélectionnez **Gestion des offres**, puis **Gérer les noms des offres**.

:::image type="content" source="media/power-bi-visual/manage-product-names.png" alt-text="Montre l’onglet de gestion des noms de produits dans une offre de visuel Power BI.":::

## <a name="reserve-a-name"></a>Réserver un nom

1. Entrez un **nom** et sélectionnez **Vérifier la disponibilité** pour voir si ce nom est disponible.
1. Si le nom est disponible, sélectionnez **Réserver un nom de produit** pour l’ajouter à la liste des noms.
1. Répétez cette opération pour chaque nom que vous souhaitez réserver.

Pour supprimer un nom, sélectionnez **Supprimer**.

Pour finaliser l’envoi de votre offre, revenez à l’un des onglets précédents (par exemple, **Configuration de l’offre**) puis sélectionnez **Vérifier et publier** en haut à droite ou en bas de la page.
