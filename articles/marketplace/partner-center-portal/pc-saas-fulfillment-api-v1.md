# <a name="saas-fulfillment-apis-version-1-deprecated"></a>API de traitement SaaS version 1 (obsolète)

> [!WARNING]
> Cette version initiale de l’API de traitement SaaS est obsolète ; nous vous recommandons d’utiliser plutôt l’[API de traitement SaaS V2](./pc-saas-fulfillment-apis.md).  Cette version initiale de l'API est actuellement uniquement maintenue pour servir les serveurs de publication existants.

La documentation pour cette version de l’API n’est plus disponible. Si vous pensez que vous devez y accéder, veuillez contacter saas-api-v1@microsoft.com.

Pour tous les nouveaux développements, passez à la [version v2 de l’API](./pc-saas-fulfillment-apis.md) même si vous utilisez déjà la version v1. Cette version sera entièrement dépréciée à l’avenir.
