# <a name="configure-iot-edge-module-offer-properties"></a>Configurer les propriétés de l’offre IoT Edge module

C’est dans cette page que vous définissez les catégories utilisées pour regrouper votre offre sur Place de marché Azure, la version de votre application et les contrats associés.

## <a name="categories"></a>Catégories

Sélectionnez des catégories et des sous-catégories pour placer votre offre dans les zones de recherche appropriées sur Place de marché Azure. Veillez à décrire ultérieurement dans la description de l’offre la façon dont votre offre prend en charge ces catégories.

- Sélectionnez une catégorie principale.
- Pour ajouter une deuxième catégorie facultative (secondaire), sélectionnez le lien **+ Catégories** .
- Sélectionnez jusqu’à deux sous-catégories pour la catégorie principale et/ou secondaire. Si aucune sous-catégorie n’est applicable à votre offre, sélectionnez **Non applicable**. Utilisez Ctrl + clic pour sélectionner une deuxième sous-catégorie.

Consultez la liste complète des catégories et sous-catégories dans [Bonnes pratiques pour le référencement des offres](gtm-offer-listing-best-practices.md).

### <a name="legal"></a>Informations juridiques

Pour utiliser le **contrat standard pour la place de marché commerciale de Microsoft**, cochez la case, puis **Accepter**. Pour fournir vos propres **Conditions générales** que les clients doivent accepter avant de pouvoir essayer votre offre, entrez-les dans la zone. Vous pouvez aussi indiquer l’URL où vos conditions générales peuvent être consultées.

> [!NOTE]
> Une fois que vous avez publié une offre à l’aide du contrat standard de Microsoft, vous ne pouvez plus la modifier dans vos conditions générales personnalisées.

Si vous effectuez des modifications, sélectionnez **Enregistrer le brouillon** avant de passer à l’onglet suivant du menu de navigation gauche **Référencement de l’offre**.
