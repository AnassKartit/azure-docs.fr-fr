# <a name="configure-virtual-machine-offer-listing-details"></a>Configurer les détails d’une liste d’offres de machines virtuelles

Dans la page **Annonce de l’offre** (que vous sélectionnez dans le menu de navigation gauche de l’Espace partenaires), vous définissez les détails de l’offre, comme le nom de l’offre, la description, les liens et les contacts du support.

Le contenu du référencement de l’offre ne doit pas nécessairement être en anglais si la description de l’offre commence par la phrase « This application is available only in [langue autre que l’anglais] ». Vous pouvez également fournir un lien vers un site qui offre du contenu dans une langue autre que celle utilisée dans le contenu de l’annonce de l’offre.

## <a name="marketplace-details"></a>Détails de la Place de marché

### <a name="name"></a>Nom

Le nom saisi ici doit être descriptif, car il s'agira du titre du référencement de votre offre. Ce champ est renseigné automatiquement avec le nom que vous avez entré dans la zone **Alias d’offre** lors de la création de l’offre. Le nom :

- Peut inclure les symboles de marque déposée et de copyright.
- Doit avoir 50 caractères au maximum.
- Ne peut pas inclure d’emojis.

### <a name="search-results-summary"></a>Résumé des résultats de recherche

Fournissez un bref résumé de l’objectif ou de la fonction de votre offre à afficher sur la page des résultats de recherche de votre offre sur la Place de marché Azure.

### <a name="short-description"></a>Description courte

Fournissez une brève description de votre offre qui sera affichée sur la page des détails de votre offre sur la Place de marché Azure.

### <a name="description"></a>Description

Fournissez une description détaillée de votre offre qui sera affichée aux clients dans la vue d’ensemble du référencement sur la Place de marché Azure.

Pensez à inclure les éléments suivants dans votre description :

- Description de l’offre
- Le type d’utilisateur qui bénéficie de l’offre
- Les besoins ou problèmes auxquels l’offre répond
- Catégorie et/ou associations sectorielles sélectionnées dans l'onglet Propriétés
- Opportunités d’achats dans l’application
- Toutes les informations requises

### <a name="privacy-policy-link"></a>Lien de la politique de confidentialité

Entrez l’adresse web (URL) de la politique de confidentialité de votre organisation. Veillez à ce que votre offre respecte les lois et les réglementations en matière de confidentialité. Vous devez également publier une politique de confidentialité valide sur votre site Web.

## <a name="useful-links"></a>Liens utiles

Fournissez des liens supplémentaires relatifs aux produits et des liens d'assistance pour votre offre.

### <a name="product-information-links"></a>Liens vers les informations produit

Fournissez éventuellement des documents en ligne complémentaires à propos de votre offre. Pour ajouter un lien, sélectionnez **Ajouter un lien**, puis renseignez les champs suivants pour chaque lien ajouté :

- **Nom** : les clients verront ce nom en tant que lien sur la page des détails.
- **Lien** : Entrez une adresse web qui permet aux clients de voir votre document en ligne.

### <a name="customer-support-links"></a>Liens de support clientèle

Indiquez le(s) site(s) Web permettant aux clients de contacter votre support technique.

- Site web d'assistance Azure Global (spécifique aux clients Azure Global)
- Site web d'assistance Azure Government (spécifique aux clients Azure Government)

## <a name="contact-information"></a>Informations de contact

Fournissez les informations de contact suivantes.

### <a name="partner-support-contact"></a>Contact du support technique du partenaire

Spécifiez le **Nom**, l’**E-mail** et le **Téléphone** que les partenaires Microsoft vont utiliser quand vos clients ouvrent un ticket de support. Ces informations ne sont pas listées sur la Place de marché Azure.

### <a name="engineering-contact"></a>Contact ingénierie

Spécifiez le **Nom**, l’**E-mail** et le **Téléphone** que Microsoft doit utiliser en cas de problème avec votre offre, notamment les problèmes de certification. Ces informations ne sont pas listées sur la Place de marché Azure.

### <a name="cloud-solution-provider-program-contact"></a>Contact du programme CSP

Le cas échéant, indiquez le **nom**, l’**adresse e-mail** et le **numéro de téléphone** du contact pour les partenaires dans le programme Fournisseur de solutions Cloud (CSP) à utiliser pour tout problème professionnel et de support. Ces informations ne sont communiquées qu’aux partenaires du programme CSP et ne figurent pas sur la Place de marché Azure. Pour **Documents marketing du programme CSP**, entrez l’adresse web où les partenaires de ce programme peuvent visualiser vos documents marketing.

## <a name="marketplace-media"></a>Médias de la Place de marché

Fournissez les logos, captures d'écran et vidéos à utiliser avec votre offre. Ces éléments apparaîtront dans les résultats de recherche et les pages des détails sur la Place de marché Azure. Toutes les images doivent être au format PNG. Votre demande sera rejetée si les images sont floues.

>[!NOTE]
>Si vous rencontrez un problème pendant le chargement de fichiers, vérifiez que votre réseau local ne bloque pas le service https://upload.xboxlive.com utilisé par l’Espace partenaires.

### <a name="logos"></a>Logos

Fournissez un fichier PNG pour le logo de **Grande taille**. L'Espace partenaires l'utilisera pour créer un logo de **Petite taille**  et un logo de **Taille moyenne**. Plus tard, vous pourrez éventuellement les remplacer par d'autres images. Ces logos sont utilisés à différents emplacements dans la liste :

- **Grande** (de 216 x 216 à 350 x 350 pixels, nécessaire) - Apparaît sur la page des détails de l’offre sur la Place de marché Azure
- **Moyenne** (90 x 90 pixels, facultatif) - Apparaît lorsque vous créez une nouvelle ressource dans le portail Azure
- **Petite** (48 x 48 pixels, facultatif) - Apparaît dans les résultats de recherche sur la Place de marché Azure

[!INCLUDE [Logo tips](includes/graphics-suggestions.md)]

### <a name="screenshots"></a>Captures d’écran.

Ajoutez éventuellement jusqu’à cinq captures d’écran qui illustrent le fonctionnement de votre offre. Incorporez les mots clés de recherche dans le nom des fichiers. Chaque capture d’écran doit avoir une taille de 1280 x 720 pixels et être au format PNG. Chaque capture d’écran doit inclure une légende. Ajoutez une légende en sélectionnant l'icône en forme de crayon qui apparaît après avoir chargé une image.

### <a name="videos"></a>Vidéos

Ajoutez jusqu’à cinq vidéos qui illustrent votre offre. Les vidéos doivent être hébergées sur un service vidéo externe (YouTube ou Vimeo uniquement). Saisissez le nom de chaque vidéo, le lien et une miniature de la vidéo sous forme d’image PNG de 1280 x 720 pixels.

Pour obtenir des ressources supplémentaires concernant la place de marché, consultez [Meilleures pratiques pour les annonces d’offre d’une place de marché](gtm-offer-listing-best-practices.md).

Sélectionnez **Enregistrer le brouillon** avant de passer à l’onglet suivant du menu de navigation gauche **Public de la préversion**.
