# <a name="switch-accounts-in-partner-center"></a>Changer de compte dans l'Espace partenaires

**Rôles appropriés**

- Propriétaire
- Manager

Vous pouvez faire partie de plusieurs comptes. Cet article explique comment déterminer si vous faites partie de plusieurs comptes et comment passer de l’un à l’autre.

## <a name="view-and-switch-accounts"></a>Afficher les comptes et en changer

[!INCLUDE [Workspaces view note](./includes/preview-interface.md)]

#### <a name="workspaces-view"></a>[Vue des espaces de travail](#tab/workspaces-view)

Pour savoir si vous faites partie de plusieurs comptes, vérifiez la présence du _sélecteur de compte_. Dans l’angle supérieur droit, sélectionnez l’icône de votre compte comme indiqué dans la capture d’écran suivante.

[ ![Illustre le sélecteur de compte dans l’Espace partenaires.](./media/manage-accounts/account-picker-workspaces.png) ](./media/manage-accounts/account-picker-workspaces.png#lightbox)

Si vous ne voyez pas le *sélecteur de compte*, vous n’êtes membre que d’un seul compte. Vous trouverez les détails de ce compte dans **Paramètres du compte** > **Profil de l’organisation** > **Légal** > **onglet Développeur** dans l’Espace partenaires.

Vous pouvez ensuite sélectionner n’importe quel compte dans la liste pour basculer vers ce compte. Après le changement, tous les éléments de l’Espace partenaires s’affichent dans le contexte de ce compte.

> [!NOTE]
> L’Espace partenaires utilise [Azure Active Directory](../active-directory/fundamentals/active-directory-whatis.md) (Azure AD) pour l’accès aux comptes multi-utilisateurs et la gestion de ceux-ci. L’Azure AD de votre organisation est automatiquement associé à votre compte Espace partenaires dans le cadre du processus d’inscription.

Dans l’exemple suivant, l’utilisateur connecté fait partie des quatre comptes en surbrillance. L’utilisateur peut passer de l’un à l’autre en cliquant sur un compte.

[ ![Capture d’écran des comptes qui peuvent être sélectionnés avec le sélecteur de compte.](./media/manage-accounts/account-picker-two-workspaces.png) ](./media/manage-accounts/account-picker-two-workspaces.png#lightbox)

#### <a name="current-view"></a>[Affichage actuel](#tab/current-view)

Pour savoir si vous êtes membre de plusieurs comptes, vérifiez la présence du *sélecteur de compte* dans le menu de navigation gauche, comme indiqué dans la capture d’écran suivante.

[ ![Capture d’écran du sélecteur de compte dans le volet de navigation de l’Espace partenaires.](./media/manage-accounts/account-picker.png) ](./media/manage-accounts/account-picker.png#lightbox)

Si vous ne voyez pas le *sélecteur de compte*, vous n’êtes membre que d’un seul compte. Vous trouverez les détails de ce compte dans **Paramètres du compte** > **Profil de l’organisation** > **Légal** > **onglet Développeur** dans l’Espace partenaires.

Lorsque vous sélectionnez ce sélecteur, tous les comptes dont vous faites partie apparaissent sous forme de liste. Vous pouvez ensuite sélectionner n’importe lequel d’entre eux pour basculer vers ce compte. Après le changement, tous les éléments de l’Espace partenaires s’affichent dans le contexte de ce compte.

> [!NOTE]
> L’Espace partenaires utilise [Azure Active Directory](../active-directory/fundamentals/active-directory-whatis.md) (Azure AD) pour l’accès aux comptes multi-utilisateurs et la gestion de ceux-ci. L’Azure AD de votre organisation est automatiquement associé à votre compte Espace partenaires dans le cadre du processus d’inscription.

Dans l’exemple suivant, l’utilisateur connecté fait partie des trois comptes en surbrillance. L’utilisateur peut passer de l’un à l’autre en cliquant sur un compte.

[ ![Capture d’écran des comptes qui peuvent être sélectionnés avec le sélecteur de compte.](./media/manage-accounts/account-picker-two.png) ](./media/manage-accounts/account-picker-two.png#lightbox)
