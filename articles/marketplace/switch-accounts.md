# <a name="switch-accounts-in-partner-center"></a>Changer de compte dans l'Espace partenaires

**Rôles appropriés**

- Propriétaire
- Manager

Vous pouvez faire partie de plusieurs comptes. Pour savoir si vous êtes membre de plusieurs comptes, vérifiez la présence du *sélecteur de compte* dans le menu de navigation gauche, comme indiqué dans la capture d’écran suivante.

[ ![Capture d’écran du sélecteur de compte dans le volet de navigation de l’Espace partenaires.](./media/manage-accounts/account-picker.png) ](./media/manage-accounts/account-picker.png#lightbox)

Si vous ne voyez pas le *sélecteur de compte*, vous n’êtes membre que d’un seul compte. Vous trouverez les détails de ce compte dans **Paramètres du compte** > **Profil de l'organisation** > **Légal** > **onglet Développeur** dans l’Espace partenaires.

Lorsque vous sélectionnez ce sélecteur, tous les comptes dont vous faites partie apparaissent sous forme de liste. Vous pouvez ensuite sélectionner n’importe lequel d’entre eux pour basculer vers ce compte. Après le changement, tous les éléments de l’Espace partenaires s’affichent dans le contexte de ce compte.

> [!NOTE]
> L’Espace partenaires utilise [Azure Active Directory](/azure/active-directory/fundamentals/active-directory-whatis) (Azure AD) pour l’accès aux comptes multi-utilisateurs et la gestion de ceux-ci. L’Azure AD de votre organisation est automatiquement associé à votre compte Espace partenaires dans le cadre du processus d’inscription.

Dans l’exemple suivant, l’utilisateur connecté fait partie des trois comptes en surbrillance. L’utilisateur peut passer de l’un à l’autre en cliquant sur un compte.
