# <a name="license-dashboard-in-commercial-marketplace-analytics"></a>Tableau de bord Licence dans l’analytique de la place de marché commerciale

Cet article fournit des informations sur le tableau de bord Licence dans le programme du marketplace commercial dans Espace partenaires. Le tableau de bord Licence affiche les informations suivantes :

- nombre de clients qui ont acheté des licences
- nombre total de licences achetées
- nombre total de licences déployées
- nombre de licences achetées et déployées par le client
- distribution de licences dans les pays et les régions

## <a name="check-license-usage"></a>Vérifier l’utilisation des licences

Pour vérifier l’utilisation des licences des applications ISV dans Espace partenaires, procédez comme suit :
1. Connectez-vous à l’[Espace partenaires](https://go.microsoft.com/fwlink/?linkid=2165507).
1. Dans le menu de navigation de gauche, sélectionnez **Marketplace commercial** > **Analyser** > **Licence**.

:::image type="content" source="./media/license-dashboard/license-dashboard.png" alt-text="Capture d’écran du tableau de bord Licence dans Espace partenaires.":::

## <a name="elements-of-the-license-dashboard"></a>Éléments du tableau de bord Licence

Les sections suivantes expliquent comment utiliser le tableau de bord Licence et lire les données.

## <a name="month-range"></a>Plage de mois

Une sélection de plages de mois est disponible en haut à droite de la page. Personnalisez la sortie des widgets sur la page en sélectionnant la plage des 6 ou 12 derniers mois, ou une plage de mois personnalisée d’une durée maximale de 12 mois. La plage de mois par défaut (période de calcul) est de six mois.

:::image type="content" source="./media/license-dashboard/month-range.png" alt-text="Capture d’écran des sélections de plages de mois dans le tableau de bord Licence dans Espace partenaires.":::

## <a name="customers-widget"></a>Widget Clients

Le _widget Clients_ affiche le nombre actuel de clients. Le graphique de tendances montre le nombre de clients d’un mois sur l’autre.

:::image type="content" source="./media/license-dashboard/customers-widget.png" alt-text="Capture d’écran du widget Clients dans le tableau de bord Licence dans Espace partenaires.":::

## <a name="license-widget"></a>Widget Licence

Le _widget Licence_ indique le nombre actuel de licences approvisionnées et attribuées. Le graphique de tendances affiche le nombre de licences approvisionnées et attribuées d’un mois sur l’autre. Vous pouvez afficher les données des 6 ou 12 derniers mois. Vous pouvez également choisir une plage de dates personnalisée en sélectionnant le lien **Personnalisé** en haut à droite de la page.

:::image type="content" source="./media/license-dashboard/license-widget.png" alt-text="Capture d’écran du widget Licence dans le tableau de bord Licence dans Espace partenaires.":::

## <a name="analysis-widget"></a>Widget Analyse

Le _widget Analyse_ montre le nombre et le pourcentage de licences approvisionnées et attribuées par offre et par plan. Le graphique de tendances affiche le nombre de licences approvisionnées et attribuées d’un mois sur l’autre. À l’aide de ce widget, vous pouvez filtrer les données par client ou par produit.

:::image type="content" source="./media/license-dashboard/analysis-widget.png" alt-text="Capture d’écran du widget Analyse dans le tableau de bord Licence dans Espace partenaires.":::

## <a name="license-distribution-widget"></a>Widget Distribution des licences

Le _widget Distribution des licences_ montre la distribution des licences dans les différents pays et régions. Les régions colorées indiquent où sont distribuées les licences utilisateur. Pour revenir à l’affichage par défaut, sélectionnez le bouton **Réinitialiser le zoom** (icône Accueil) dans le widget.

:::image type="content" source="./media/license-dashboard/license-distribution.png" alt-text="Capture d’écran du widget Distribution des licences dans le tableau de bord Licence dans Espace partenaires.":::

## <a name="data-terms-in-license-report-downloads"></a>Conditions générales des données dans Téléchargements des rapports de licence

Vous pouvez utiliser l’icône de téléchargement située dans l’angle supérieur droit d’un widget pour télécharger les données.

| Nom de l’attribut | Définition |
| ------------ | ------------- |
| Pays du client | Pays de facturation du client |
| Code du pays du client | Code du pays de facturation du client |
| Nom du client | Nom de client |
| Date d’activation | Date à laquelle les licences ont été activées |
| Nom d’affichage du produit | Titre de l’offre tel qu’il apparaît dans AppSource |
| Product ID | Product ID |
| Licences approvisionnées | Nombre de licences activées dans le compte du client |
| Licences attribuées | Nombre de licences que le client a attribuées à ses utilisateurs |
| Nom de la référence SKU | Nom du plan dans l’offre |
| ID client | ID unique du locataire |
| État de la licence | État de la licence |
| ID de service | Identificateur unique utilisé dans le package pour faire correspondre le plan aux vérifications de licence |
|||
