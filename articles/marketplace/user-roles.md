# <a name="assign-user-roles-and-permissions"></a>Attribuer des rôles et autorisations d’utilisateur

**Rôles appropriés**

- Propriétaire
- Manager

Afin d’accéder aux capacités relatives au marketplace ou à votre compte de développeur, vous devez vous voir attribuer un ou plusieurs des rôles suivants. Chaque rôle dispose d’un ensemble spécifique d’autorisations permettant d’exécuter certaines fonctions au sein du compte.

| Role | Description |
| ------------ | ------------- |
| Propriétaire | Lorsqu’un compte d’éditeur est créé, un rôle de propriétaire est attribué au partenaire.<br><br>Détails des autorisations :<br><br>&#10004;&#160;&#160;Propriété complète du compte<br>&#10004;&#160;&#160;Ajouter un type d’utilisateurs<br>&#10004;&#160;&#160;Peut renseigner les informations relatives aux paiements et aux taxes<br>&#10004;&#160;&#160;Afficher les rapports et les résumés de commentaires<br>&#10004;&#160;&#160;Inscrire le compte d’éditeur dans tout autre programme de développeur |
| Manager | &#10004;&#160;&#160;Gérer les utilisateurs, les rôles et les autorisations du compte d’éditeur<br>&#10004;&#160;&#160;Gérer les locataires<br>&#x2718;&#160;&#160;Accéder aux paramètres des taxes et des paiements |
| Développeur | &#10004;&#160;&#160;Charger le package<br>&#10004;&#160;&#160;Envoyer des applications et des modules complémentaires<br>&#10004;&#160;&#160;Afficher les rapports d’utilisation des détails de télémétrie<br>&#x2718;&#160;&#160;Accéder aux informations financières ou aux paramètres de compte |
| Contributeur commercial | &#10004;&#160;&#160;Accéder aux informations financières<br>&#10004;&#160;&#160;Définir les détails de la tarification<br>&#x2718;&#160;&#160;Créer ou soumettre de nouvelles applications et de nouveaux modules complémentaires |
| Contributeur financier | &#10004;&#160;&#160;Afficher les rapports de paiement<br>&#10004;&#160;&#160;Gérer les profils de paiement et fiscaux<br>&#x2718;&#160;&#160;Modifier les applications ou les paramètres |
| Distributeur | &#10004;&#160;&#160;Répondre aux évaluations des clients<br>&#10004;&#160;&#160;Afficher des rapports non financiers<br>&#x2718;&#160;&#160;Modifier les applications ou les paramètres |
|||

> [!NOTE]
> Dans le cadre du programme Place de marché commerciale, les rôles Administrateur général, Contributeur professionnel, Contributeur financier et Responsable marketing ne sont pas utilisés. L'attribution de ces rôles aux utilisateurs n'a aucun effet. Seuls les rôles Manager et Développeur accordent des autorisations aux utilisateurs.

Pour plus d'informations sur la gestion des rôles et des autorisations dans le cadre d'autres programmes de l'Espace partenaires (Azure Active Directory (AD), Fournisseur de solutions Cloud (CSP), Fournisseur de panneau de contrôle (CPV), Utilisateurs invités, Microsoft Partner Network (MPN), etc.), consultez [Attribuer des rôles et des autorisations aux utilisateurs dans l'Espace partenaires](/partner-center/permissions-overview).

> [!NOTE]
> Toutes les activités de gestion des utilisateurs et d’attribution des rôles effectuées sur ces lignes se feront dans le compte sur lequel vous vous trouvez. Reportez-vous à la section sur le passage d’un vendeur à l’autre si vous devez gérer un autre compte.
