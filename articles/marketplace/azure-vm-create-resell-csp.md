# <a name="resell-your-offer-through-csp"></a>Revente d’une offre par l’intermédiaire d’un fournisseur CSP

L’activation de cette option permet aux partenaires qui font partie du programme Fournisseur de solutions cloud (CSP) de Microsoft de revendre votre machine virtuelle dans le cadre d’une offre groupée. Tous les plans BYOL (apportez votre propre licence) rejoignent automatiquement le programme. Vous pouvez également choisir de joindre vos plans non BYOL. Pour plus d’informations, consultez [Programme des fournisseurs de solution cloud](cloud-solution-providers.md).

## <a name="resell-through-csp"></a>Revendre via un CSP

Sur la page **Revendre via des fournisseurs de solutions Cloud** de votre offre, vous pouvez étendre la portée de votre offre en la mettant à la disposition des partenaires du programme [Fournisseur de solutions Cloud (CSP)](https://azure.microsoft.com/offers/ms-azr-0145p/).

1. Sélectionnez les partenaires qui peuvent revendre votre offre :

    - Tous les partenaires du programme CSP
    - Partenaires spécifiques du programme CSP que je sélectionne
    - Aucun partenaire du programme CSP

1. Sélectionnez **Enregistrer le brouillon** avant de continuer.
1. Si vous avez choisi de proposer une version d’évaluation avec cette offre, dans le menu de navigation de gauche, sélectionnez **Version d’évaluation**. Sinon, sélectionnez **Vérifier et publier**.
