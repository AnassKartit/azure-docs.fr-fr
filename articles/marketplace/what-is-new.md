# <a name="whats-new-in-the-microsoft-commercial-marketplace"></a>Nouveautés de la place de marché commerciale Microsoft

Découvrez les mises à jour importantes dont a fait l’objet le programme de place de marché commerciale de l’Espace partenaires. Cette page étant fréquemment mise à jour, consultez-la régulièrement !

## <a name="new-features"></a>Nouvelles fonctionnalités

| Category | Description | Date |
| --- | --- | --- |
| Analytique | Les éditeurs d’offres d’application Azure peuvent consulter l’intégrité du déploiement des offres dans les rapports de Qualité de service (QoS). La QoS aide les éditeurs à comprendre les raisons pour lesquelles le déploiement des offres échoue et fournit des informations exploitables pour y remédier. Pour plus d’informations, consultez [Tableau de bord de la Qualité de service (QoS)](quality-of-service-dashboard.md). | 07-09-2021 |
| Offres | L’installation et la maintenance des visuels Power BI migrent d’Office Store vers le marketplace commercial au cours des prochaines semaines. [Cette FAQ](/azure/marketplace/power-bi-visual-faq) fournit un résumé des améliorations apportées au processus de soumission des offres. Pour commencer, consultez [Planifier une offre de visuel Power BI](marketplace-power-bi-visual.md).| 2021-09-21 |
| Offres | Des propriétés supplémentaires au niveau du plan sont désormais disponibles pour les offres de machines virtuelles Azure. Pour plus d’informations, consultez l’article [Propriétés de la configuration technique des machines virtuelles](azure-vm-create-plans.md#properties). | 26-07-2021 |
| Frais | Microsoft a changé les frais de certains services. Consultez [Fonctionnalités de transaction sur la place de marché commerciale](marketplace-commercial-transaction-capabilities-and-considerations.md#examples-of-pricing-and-store-fees) et les questions courantes sur les paiements et les taxes, « [Comment trouver les frais de service de magasin actuels et le taux de paiement ?](/partner-center/payout-faq) ». | 14-07-2021 |
| Offres | Les éditeurs peuvent publier une machine virtuelle qu’ils ont créée localement. Pour en savoir plus, consultez [Créer une machine virtuelle à l’aide de votre propre image](./azure-vm-create-using-own-image.md). | 23-06-2021 |
| Engagement client | Les éditeurs peuvent désormais répondre aux avis des utilisateurs sur la Place de marché Azure ou AppSource. Pour en savoir plus, consultez [Tableau de bord analytique Évaluations et avis dans l’Espace partenaires](./ratings-reviews.md). | 03-06-2021 |
| Offres | Les éditeurs disposent désormais d’un moyen plus simple et plus rapide de préparer et publier leurs offres basées sur des machines virtuelles Azure dans l’Espace partenaires. Pour plus d’informations, consultez [Création d’une machine virtuelle à l’aide d’une base approuvée](azure-vm-create-using-approved-base.md). | 2021-03-22 |
| Analytics | Les développeurs peuvent utiliser de nouvelles API de rapport pour accéder programmatiquement aux données analytiques de la place de marché commerciale. Vous pouvez planifier des rapports personnalisés et télécharger vos données de la place de marché dans vos systèmes analytiques internes. Pour plus d’informations, consultez [Bien démarrer avec l’accès programmatique aux données analytiques](analytics-get-started.md). | 08-03-2021 |
|

## <a name="tax-updates"></a>Mises à jour fiscales

| Category | Description | Date |
| --- | --- | --- |
| Taxes | La taxation sur les clients finaux en Australie est gérée par Microsoft, à l’exception des achats que font les clients dans le cadre d’un contrat Entreprise, lesquels sont gérés par l’éditeur. | 01-07-2021 |
| Taxes | Mise à jour de la liste des pays dans la [page de détails des taxes](/partner-center/tax-details-marketplace) pour inclure les éléments suivants : <ul><li>Argentine</li><li>Bulgarie</li><li>Hong Kong (R.A.S.)</li><li>Corée du Sud</li><li>Pakistan</li><li>Autorité palestinienne</li><li>Panama</li><li>Paraguay</li><li>Pérou</li><li>Philippines</li><li>Saint-Kitts-et-Nevis</li><li>Sénégal</li><li>Sri Lanka</li><li>Tadjikistan</li><li>Tanzanie</li><li>Thaïlande</li><li>Trinité-et-Tobago</li><li>Tunisie</li><li>Turkménistan</li><li>Ouganda</li><li>Ouzbékistan</li><li>Zimbabwe</li></ul> | 01-07-2021 |
| Taxes | Le Nigeria a été déplacé de la liste « Pays où l’éditeur et le développeur se partagent la gestion » vers « Taxation sur les clients finaux qui varie selon les places de marché ».  | 01-07-2021 |
| Paiements | Ajout de scénarios de paiement de la place de marché commerciale à la [page de détails des taxes](/partner-center/tax-details-marketplace). | 26-04-2021 |
