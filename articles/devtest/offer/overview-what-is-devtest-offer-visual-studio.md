# <a name="what-is-the-azure-devtest-offer"></a>Qu’est-ce que l’offre Dev/Test Azure ?

Cet article explique l’offre Dev/Test Azure faisant partie des avantages de votre abonnement Visual Studio. Avec cette offre, vous bénéficiez de crédits mensuels Azure pour explorer et essayer différents services Azure gratuitement.  Les montants du crédit varient selon le niveau de votre abonnement.  Les utilisateurs peuvent également bénéficier de tarifs dev/test Azure réduits pour divers services Azure, ce qui permet un développement et des tests économiques.

## <a name="who-can-benefit"></a>Qui peut en bénéficier ? 

Avec l’offre Dev/Test Azure, les abonnés Visual Studio suivants peuvent utiliser Azure :
+ [Des particuliers peuvent utiliser Azure comme bac à sable personnel pour le développement/test](quickstart-individual-credit.md). 
+ [Des entreprises peuvent utiliser Azure pour le développement et la collaboration en équipe](quickstart-create-enterprise-devtest-subscriptions.md).
