# <a name="changing-azure-devtest-offers--remove-limits"></a>Changer d’offre Azure Dev/Test et supprimer les limites

Si vous disposez actuellement d’un abonnement aux tarifs du paiement à l’utilisation, vous pouvez basculer votre abonnement Azure vers une autre offre. Les raisons qui vous poussent à changer d’offre peuvent être spécifiques à la structure de votre organisation. Mais voici quelques raisons courantes pour lesquelles vous pourriez vouloir changer votre offre d’abonnement ou supprimer les limites de crédit.  

- La limite de crédit vous permet d’exécuter des expériences, des tests, etc. sans débiter votre carte de crédit. Lors de l’exécution de services, la facture peut vite monter. La limite de crédit place des garde-fous autour de votre abonnement. Vous pouvez créer l’architecture de votre choix et l’exécuter aussi longtemps que nécessaire sans dépenser trop.  
- Une collaboration avec d’autres organisations ou des changements au sein de votre organisation peuvent vous obliger à changer de type d’offre.  
- Vos besoins en matière d’abonnement Azure peuvent évoluer de telle sorte que vous devez continuer à modifier le type d’offre actuel. Vous pouvez également être amené à modifier la façon dont vous travaillez sur un projet. Dans ce cas, vous devrez peut-être passer d’une structure de limite de crédit à une structure de paiement à l’utilisation pour exécuter les services dont vous avez besoin.  
- Vous avez un projet critique à mener et avez besoin de plus de temps. Vous pouvez supprimer les limites si vous avez besoin d’un peu plus que la limite de 50 USD ou 150 USD.  
- Votre offre actuelle a expiré.  

Changer votre offre d’abonnement est facile pour certains types d’offre. Vous pouvez suivre [notre documentation sur le changement d’offre Azure](../../cost-management-billing/manage/switch-azure-offer.md) ou contacter le support pour obtenir de l’aide.  

## <a name="understand-spending-limits-and-cost-management"></a>Comprendre les limites de dépense et la gestion des coûts

La [gestion des coûts](../../cost-management-billing/costs/cost-mgt-best-practices.md) des abonnements à crédit mensuel Azure vous fournit les outils nécessaires pour planifier, analyser et réduire vos dépenses en vue d’optimiser votre investissement dans le cloud. Ces outils vous donnent la possibilité de supprimer et de rajouter vos limites de dépense.  

La limite de dépense dans Azure empêche les dépenses excessives. Bien que vous ne puissiez pas modifier la limite de dépense, vous pouvez la supprimer. Soit vous n’aurez aucune limite, soit vous aurez une limite égale au montant du crédit.  

Vous pouvez [supprimer la limite de dépense](../../cost-management-billing/manage/spending-limit.md) à tout moment tant qu’un mode de paiement valide est associé à votre abonnement Azure.  

Plus de documents sur la gestion des coûts :  

* [Comprendre les données Cost Management](../../cost-management-billing/costs/understand-cost-mgt-data.md)  
* [Optimiser votre investissement dans le cloud avec Azure Cost Management](../../cost-management-billing/costs/cost-mgt-best-practices.md)  

## <a name="payment-methods-and-overages"></a>Modes de paiement et dépassements

Lorsque vous supprimez des limites ou changez d’offre, il peut vous être demandé de[ mettre à jour le crédit ou d’en ajouter un](../../cost-management-billing/manage/change-credit-card.md). Vous ne pouvez pas utiliser votre carte de crédit pour les dépassements tant que votre crédit n’est pas épuisé.  

Dans le portail Azure, vous pouvez remplacer votre mode de paiement par défaut par une nouvelle carte de crédit et mettre à jour les informations de votre carte de crédit. Vous devez être [Administrateur de compte](../../cost-management-billing/understand/subscription-transfer.md#whoisaa) ou disposer des [autorisations MCA](../../cost-management-billing/manage/understand-mca-roles.md) appropriées pour effectuer ces modifications.  

Si vous voulez supprimer une carte de crédit, consultez [Supprimer un mode de paiement pour la facturation Azure](../../cost-management-billing/manage/delete-azure-payment-method.md).  

Vous pouvez également mettre à jour votre mode de paiement en remplaçant la carte de crédit par un chèque ou un virement bancaire. Vous devez d’abord obtenir l’autorisation de payer par facture. Pour obtenir l’autorisation, consultez [Paiement des abonnements Azure par facture](../../cost-management-billing/manage/pay-by-invoice.md).
