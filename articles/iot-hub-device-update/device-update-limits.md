# <a name="device-update-for-iot-hub-limits"></a>Limites Device Update pour IoT Hub

Ce document fournit une vue d’ensemble des différentes limites qui sont imposées sur la ressource Device Update pour IoT Hub et les opérations qui lui sont associées. Il indique également si les limites sont ajustables, ou non, en contactant le Support Microsoft. 

## <a name="preview-limits"></a>Limites de la préversion

Pendant la préversion, le service Device Update pour IoT Hub est fourni gratuitement aux clients. Des limites plus restrictives sont imposées lors de l’offre de préversion du service. Ces limites sont supposées changer une fois que le service est mis en disposition générale.
