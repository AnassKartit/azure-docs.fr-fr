# <a name="azure-policy-guest-configuration-baseline-for-windows"></a>Base de référence de configuration invité de Azure Policy pour Windows

L’article suivant détaille les conditions requises par les machines Windows **\[en préversion\] pour les audits d’initiative de la stratégie de configuration d’invités de la base de référence de sécurité Azure**.
