# <a name="deploy-an-azure-app-service-app-by-using-visual-studio-2015"></a>Déployer une application Azure App Service avec Visual Studio 2015

[!INCLUDE [closureinfo](../../includes/germany-closure-info.md)]

Cet article explique comment déployer une application Azure App Service (application API, application web ou application mobile) sur Azure Germany à l’aide de Visual Studio 2015.

## <a name="prerequisites"></a>Configuration requise
* Consultez la [configuration requise par Visual Studio](../app-service/quickstart-dotnetcore.md#prerequisites) pour savoir comment installer et configurer Visual Studio 2015 et le Kit de développement logiciel (SDK).
* Suivez [ces instructions](./germany-get-started-connect-with-vs.md) pour configurer Visual Studio pour une connexion à un compte Azure Germany. 

## <a name="open-an-app-project-in-visual-studio"></a>Ouvrir un projet d’application dans Visual Studio
Ouvrez une solution d’application ou un projet existants dans Visual Studio ou [créez un projet](../app-service/quickstart-dotnetcore.md?tabs=netframework48#create-an-aspnet-web-app). Ensuite, exécutez l’application dans Visual Studio pour vérifier qu’elle fonctionne en local.

## <a name="deploy-to-azure-germany"></a>Déploiement sur Azure Germany
Une fois que Visual Studio est configuré pour se connecter à votre compte Azure Germany (opération préalable requise), les [instructions permettant de déployer une application App Service](../app-service/quickstart-dotnetcore.md) sont exactement les mêmes que celles qui concernent Azure global.
