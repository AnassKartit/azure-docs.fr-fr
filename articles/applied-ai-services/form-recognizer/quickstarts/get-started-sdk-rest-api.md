# <a name="get-started-with-a-client-library-sdks-or-rest-api"></a>Prise en main des SDK de la bibliothèque de client ou de l’API REST

Commencez avec Azure Form Recognizer dans le langage de programmation de votre choix. Azure Form Recognizer est un service Azure Applied AI Service qui utilise le machine learning pour extraire et analyser les champs de formulaire, le texte et les tableaux de vos documents. Vous pouvez facilement appeler des modèles Form Recognizer en intégrant les kits SDK de notre bibliothèque de client dans vos workflows et applications. Nous vous recommandons d’utiliser le service gratuit pendant que vous apprenez la technologie. N’oubliez pas que le nombre de pages gratuites est limité à 500 par mois.

Pour en savoir plus sur les fonctionnalités Form Recognizer les options de développement, visitez notre page de [présentation](../overview.md#form-recognizer-features-and-development-options).
