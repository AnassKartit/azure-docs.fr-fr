# <a name="security-posture--cis-benchmark"></a>Posture de sécurité – Benchmark CIS 

Le micro-agent Defender for IoT permet aux organisations d’améliorer leur conformité et leur posture de sécurité. Les recommandations de base du benchmark CIS pour le système d’exploitation permettent d’identifier les problèmes liés à la gestion de la sécurité des appareils et de classer par ordre de priorité les modifications pour la sécurisation renforcée.  

## <a name="best-practices-for-secure-configuration"></a>Meilleures pratiques en matière de configuration sécurisée

Les benchmarks CIS constituent les meilleures pratiques pour sécuriser la configuration d’un système cible. Ils sont développés suivant un processus unique établi par consensus entre des professionnels de la cybersécurité et des experts techniques dans le monde entier. 

Les benchmarks CIS sont les seuls guides de configuration de la sécurité développés et acceptés par les administrations, les entreprises, l’industrie et le milieu universitaire.

Le micro-agent Azure Defender pour IoT vous permet d’améliorer rapidement les fonctionnalités de protection et de sécurité des appareils de votre organisation en proposant les meilleures pratiques de configuration CIS, ainsi qu’une identification constante de tous les liens faibles existants dans la posture de sécurité de votre système d’exploitation.
