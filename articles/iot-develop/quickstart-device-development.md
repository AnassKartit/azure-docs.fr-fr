# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Prise en main du développement d’appareils intégrés Azure IoT

**S’applique à** : [Développement d’appareils intégrés](about-iot-develop.md#embedded-device-development)

Ce guide de prise en main contient un ensemble de démarrages rapides qui vous montrent comment commencer à travailler avec des appareils intégrés et Azure IoT. 

Dans chaque démarrage rapide, vous effectuez les tâches de base suivantes :
* Installer un ensemble d’outils de développement intégrés pour la programmation d’un appareil spécifique en C
* Créer une image qui comprend des composants et des exemples d’Azure RTOS, puis flasher un appareil
* Connecter en toute sécurité un appareil à Azure IoT
* Afficher la télémétrie de l’appareil, afficher les propriétés et appeler les méthodes cloud-à-appareil
