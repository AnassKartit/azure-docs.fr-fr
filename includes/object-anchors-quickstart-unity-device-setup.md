## <a name="set-up-your-device"></a>Configurer votre appareil

Pour déployer des applications sur votre HoloLens, vous devez associer votre HoloLens à votre ordinateur.

1. Dans votre HoloLens, accédez à **Paramètres -> Mettre à jour et sécurité -> Pour les développeurs**.
2. Cliquez sur **Associer** et laissez l’écran ouvert jusqu’à ce que vous entriez le code confidentiel dans Visual Studio lors de votre premier déploiement.
