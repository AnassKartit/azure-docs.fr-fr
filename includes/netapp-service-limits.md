Azure NetApp Files a une limite régionale pour la capacité. La limite de capacité standard pour chaque abonnement est de 25 Tio, par région, pour tous les niveaux de service. Pour augmenter la capacité, recourrez à la demande de support [ Limites du service et de l’abonnement (quotas)](../articles/azure-netapp-files/azure-netapp-files-resource-limits.md#request-limit-increase-).

Pour en savoir plus sur les limites des Azure NetApp Files, consultez [Limites des ressources pour les Azure NetApp Files](../articles/azure-netapp-files/azure-netapp-files-resource-limits.md).
