### <a name="migrate-databases-with-azure-sql-migration-extension-for-azure-data-studio-preview"></a>Migrer des bases de données avec l’extension Migration Azure SQL pour Azure Data Studio (préversion)
Vous pouvez utiliser l’[extension Migration Azure SQL dans Azure Data Studio](/sql/azure-data-studio/extensions/azure-sql-migration-extension) pour migrer des bases de données SQL Server vers Azure SQL Managed Instance (Platform-as-a-service) ou vers SQL Server sur des machines virtuelles Azure (Infrastructure-as-a-service). L’extension Migration Azure SQL pour Azure Data Studio fournit un Assistant qui évalue vos bases de données SQL Server pour la migration vers Azure SQL Managed Instance ou SQL Server sur des machines virtuelles Azure, puis les migre en choisissant un mode de migration en ligne ou hors connexion. </br>
Azure Database Migration Service orchestre les activités de déplacement des données et supervise les activités de migration.
<br/>
Pour en savoir plus, consultez [Migrer des bases de données avec l’extension Migration Azure SQL pour Azure Data Studio](../articles/dms/migration-using-azure-data-studio.md)
