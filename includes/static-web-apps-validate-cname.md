#### <a name="validate-cname"></a>Valider CNAME

1. Revenez à la fenêtre _Valider + ajouter_ dans le portail Azure.

1. Sélectionnez le bouton **Ajouter**.

Azure tente de valider le nouveau CNAME auprès de votre fournisseur de domaine. L’opération peut prendre quelques minutes en fonction du fournisseur de domaine. Si la validation échoue immédiatement, patientez quelques minutes et réessayez avant de procéder à une résolution des problèmes.

À présent que le sous-domaine est configuré, il peut falloir plusieurs heures au fournisseur de DNS pour propager les modifications dans le monde entier.
