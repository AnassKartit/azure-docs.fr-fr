# <a name="mixed-reality-feature-tool"></a>[Mixed Reality Feature Tool](#tab/unity-package-mixed-reality-feature-tool)

Cette étape nécessite que <a href="/windows/mixed-reality/develop/unity/welcome-to-mr-feature-tool" target="_blank">Mixed Reality Feature Tool</a> soit téléchargé et disponible.

> [!NOTE]
> <a href="/windows/mixed-reality/develop/unity/welcome-to-mr-feature-tool" target="_blank">Mixed Reality Feature Tool</a> ne prend actuellement en charge que Windows.

1. Suivez la documentation de <a href="/windows/mixed-reality/develop/unity/welcome-to-mr-feature-tool" target="_blank">Mixed Reality Feature Tool</a> pour configurer l’outil et découvrir comment l’utiliser.
2. Installez le package principal du SDK Azure Spatial Anchors, ainsi que tous les packages propres à la plateforme, dans le dossier du projet Unity.
