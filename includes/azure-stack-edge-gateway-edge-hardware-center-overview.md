Pour approvisionner un appareil, accédez à Azure Edge Hardware Center (préversion) et passez une commande. Azure Edge Hardware Center vous permet de choisir parmi une large gamme de références SKU Azure Stack, pour répondre aux besoins de votre entreprise. Vous pouvez commander plusieurs unités d’un type d’appareil, expédier plusieurs appareils à différents emplacements, enregistrer des adresses pour des commandes ultérieures et suivre l’état de vos commandes. 

Une fois la commande remise, vous pouvez configurer votre appareil et créer une ressource Azure Stack Edge pour gérer l’appareil.
