> [!IMPORTANT]
> Si l’avertissement suivant s’affiche, vous devez mettre à jour le logiciel sur les appareils avant de continuer :<br>
> 
> *Un ou plusieurs appareils StorSimple exécutent une version logicielle antérieure. La dernière mise à jour disponible pour TLS 1.2 est une mise à jour obligatoire et doit être installée immédiatement sur ces appareils. TLS 1.2 est utilisé pour toutes les communications du portail Azure, et sans cette mise à jour, l’appareil ne peut pas communiquer avec le service StorSimple.*
>
