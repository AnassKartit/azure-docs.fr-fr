## <a name="developer-interfaces"></a>Interfaces développeur

Cet article montre comment effectuer différentes opérations de gestion à l’aide du [kit SDK .NET (C#)](/dotnet/api/overview/azure/digitaltwins/management?view=azure-dotnet&preserve-view=true). Vous pouvez également créer les mêmes appels de gestion à l’aide des autres kits de langage SDK décrits dans les [API Azure Digital Twins et kits SDK](../articles/digital-twins/concepts-apis-sdks.md).

Les autres interfaces développeur qui peuvent être utilisées pour effectuer ces opérations sont les suivantes :
* [Azure Digital Twins Explorer](../articles/digital-twins/concepts-azure-digital-twins-explorer.md)
* Appels de l’[API REST](/rest/api/azure-digitaltwins/)
* Commandes [Azure CLI](/cli/azure/dt?view=azure-cli-latest&preserve-view=true)
