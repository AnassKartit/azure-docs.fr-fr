***Avant de commencer***, utilisez le sélecteur ci-dessus pour choisir le type de stratégie que vous configurez. Azure AD B2C propose deux moyens de définir la façon dont les utilisateurs interagissent avec vos applications : [flux utilisateurs](../articles/active-directory-b2c/user-flow-overview.md) prédéfinis ou [stratégies personnalisées](../articles/active-directory-b2c/custom-policy-overview.md) entièrement configurables. La procédure donnée dans cet article est différente pour chaque méthode.
