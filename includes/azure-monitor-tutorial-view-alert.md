Quand une alerte se déclenche, elle envoie des notifications dans ses groupes d’actions. Vous pouvez également afficher l’alerte dans le portail Azure. 

Sélectionnez **Alertes** dans le menu de la ressource. S’il existe des alertes ouvertes pour les ressources, elles sont incluses dans l’affichage.

:::image type="content" source="./media/azure-monitor-tutorial-view-alert/alerts-view.png" lightbox="./media/azure-monitor-tutorial-view-alert/alerts-view.png" alt-text="Affichage des alertes":::

Cliquez sur une gravité pour voir les alertes correspondantes. Cochez **État de l’alerte** et décochez **Fermé** pour afficher uniquement les alertes ouvertes.

:::image type="content" source="./media/azure-monitor-tutorial-view-alert/alert-state.png" lightbox="./media/azure-monitor-tutorial-view-alert/alert-state.png" alt-text="Filtre de l’état de l’alerte":::

Cliquez sur le nom d’une alerte pour voir ses détails.

:::image type="content" source="./media/azure-monitor-tutorial-view-alert/alert-detail.png" lightbox="./media/azure-monitor-tutorial-view-alert/alert-detail.png" alt-text="Détails de l’alerte":::
