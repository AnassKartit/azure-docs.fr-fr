> [!NOTE]
> Les points de terminaison Azure Machine Learning (préversion) offrent une expérience de déploiement plus simple et améliorée. Les points de terminaison prennent en charge les scénarios d’inférence en temps réel et par lot. Les points de terminaison fournissent une interface unifiée pour appeler et gérer des déploiements de modèle pour différents types de calcul. Consultez [Que sont les points de terminaison Azure Machine Learning (préversion) ?](../articles/machine-learning/concept-endpoints.md).
