Si le retour de l’appareil s’effectue en Corée, procédez comme suit.

1. Notez le numéro de demande de service (le *numéro de référence* dans la page **Préparation de l’expédition** de l’interface utilisateur web locale de la Data Box).
1. Vérifiez que l’appareil est hors tension et que les câbles sont débranchés. Enroulez le cordon d’alimentation fourni avec l’appareil et fixez-le dans le compartiment arrière.
1. Empaquetez l’appareil à l’aide de la boîte d’origine utilisée pour l’expédition. Veillez à inclure le bordereau de réexpédition.
1. Pour planifier une collecte, contactez l’assistance téléphonique de *Quantium Solutions International* au 070-8231-1418 pendant les heures de bureau (du lundi au vendredi de 10 h à 17 h). Indiquez *Enlèvement Microsoft Azure* et le numéro de la demande de service pour organiser l’enlèvement.

   Vous pouvez également envoyer un e-mail à [microsoft@rocketparcel.com](mailto:microsoft@rocketparcel.com) pour planifier une collecte. Utilisez le modèle suivant :

   ```
   To: microsoft@rocketparcel.com
   Subject: Microsoft Azure Pickup | Service Request Number: <ReferenceNumber>
   Body:
   * Service Request Number (reference number): <ReferenceNumber>
   * Requested pickup date: yyyy/mm/dd;time:HH MM
   * <If the consignment note is missing, note that here along with the sender and receiver information. If the consignment note was included, please delete this section.>
   ```  

1. Si vous rencontrez des problèmes ou si vous êtes invité à payer des frais supplémentaires lors de la planification d’une collecte, envoyez un e-mail aux opérations Azure Data Box à l’adresse [adbops@microsoft.com](mailto:adbops@microsoft.com) pour obtenir de l’aide. Indiquez le nom de la commande et le problème rencontré dans la ligne **Objet**.

Une fois l’appareil récupéré et analysé par le transporteur, l’état de la commande dans le portail Azure est mis à jour pour être **Enlevé** et un ID de suivi s’affiche.
