> [!IMPORTANT]
> La fonctionnalité de chargement de fichiers sur les appareils qui utilisent l’authentification par l’autorité de certification X.509 est en préversion publique, et le [mode aperçu doit être activé](../articles/iot-hub/iot-hub-preview-mode.md). Elle est généralement disponible sur les appareils qui utilisent l’authentification par empreinte X.509. Pour en savoir plus sur l’authentification X.509 avec IoT Hub, consultez les [certificats X.509 pris en charge](../articles/iot-hub/iot-hub-dev-guide-sas.md#supported-x509-certificates).
