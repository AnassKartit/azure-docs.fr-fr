> [!IMPORTANT]
> Les commandes Azure CLI de cet article __requièrent__ l’extension `azure-cli-ml`, ou v1, pour Azure Machine Learning. L’interface CLI v2 améliorée (préversion) utilisant l’extension `ml` est désormais disponible et recommandée. Les extensions étant incompatibles, les commandes de l’interface CLI v2 ne fonctionneront pas pour les étapes décrites dans cet article. Toutefois, il est possible d’interagir avec les espaces de travail Machine Learning et toutes les ressources sous-jacentes à partir de l’une ou l’autre de ces extensions, ce qui signifie qu’un utilisateur peut créer un espace de travail avec l’interface CLI v1 et un autre peut envoyer des travaux au même espace de travail avec l’interface CLI v2.
>
> Pour savoir quelles extensions vous avez installées, utilisez `az extension list`. Si la liste des __extensions__ contient `azure-cli-ml`, vous disposez de l’extension appropriée pour les étapes de cet article.
>
> Pour plus d’informations sur l’installation et l’utilisation des différentes extensions, consultez les articles suivants :
> 
> * `azure-cli-ml` - [Installer, configurer et utiliser l’interface CLI (v1)](../articles/machine-learning/reference-azure-machine-learning-cli.md)
> * `ml` - [Installer et configurer l’interface CLI (v2)](../articles/machine-learning/how-to-configure-cli.md)
