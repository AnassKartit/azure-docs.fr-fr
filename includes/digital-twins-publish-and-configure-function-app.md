Pour obtenir des instructions sur la façon de procéder, consultez la section [Publier l’application de fonction dans Azure](../articles/digital-twins/how-to-create-azure-function.md#publish-the-function-app-to-azure) de l’article *Guide pratique : configurer une fonction pour le traitement des données*.

> [!IMPORTANT]
> Lors de la création de l’application de fonction pour la première fois dans la [section Prérequis](#prerequisites), vous avez peut-être déjà attribué un rôle d’accès à la fonction et configuré les paramètres d’application pour l’accès à votre instance Azure Digital Twins. Ces opérations doivent être effectuées une seule fois pour l’ensemble de l’application de fonction. Vérifiez qu’elles ont été effectuées dans votre application avant de continuer. Vous pouvez obtenir des instructions dans la section [Configurer l’accès à la sécurité pour l’application de fonction](../articles/digital-twins/how-to-create-azure-function.md#set-up-security-access-for-the-function-app) de l’article *Guide pratique : ingérer des données IoT Hub*.
