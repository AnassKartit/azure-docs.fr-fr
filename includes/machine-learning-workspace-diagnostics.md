À partir [d’Azure Machine Learning studio](https://ml.azure.com), vous pouvez exécuter des diagnostics sur votre espace de travail pour vérifier votre configuration. Pour exécuter les diagnostics, sélectionnez l’icône «  __?__  » dans le coin supérieur droit de la page. Sélectionnez ensuite __Exécuter les diagnostics de l’espace de travail__.

:::image type="content" source="./media/machine-learning-workspace-diagnostics/diagnostics.png" alt-text="Capture d’écran du bouton Diagnostics de l’espace de travail":::

Une fois les diagnostics exécutés, une liste de tous les problèmes détectés est retournée. Cette liste contient des liens vers des solutions possibles.
