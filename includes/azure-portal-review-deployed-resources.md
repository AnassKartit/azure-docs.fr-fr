Vérifiez que vous pouvez voir les données sur la machine virtuelle à partir du tableau de bord du portail Azure.

1. Dans le portail Azure, sélectionnez **Tableau de bord**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/navigate-to-dashboards.png" alt-text="Navigation du portail Azure jusqu’au tableau de bord":::

1. Dans la page Tableau de bord, sélectionnez **Tableau de bord Machine virtuelle simple**.

    :::image type="content" source="media/azure-portal-review-deployed-resources/select-simple-vm-dashboard.png" alt-text="Accéder au tableau de bord Machine virtuelle simple":::

1. Examinez le tableau de bord créé par le modèle Azure Resource Manager (modèle ARM). Vous pouvez voir qu’une partie du contenu est statique, mais il y a également des graphiques qui indiquent les performances de la machine virtuelle que vous avez créée au début.

    :::image type="content" source="media/azure-portal-review-deployed-resources/review-simple-vm-dashboard.png" alt-text="Examiner le tableau de bord Machine virtuelle simple":::
