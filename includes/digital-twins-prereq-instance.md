Pour utiliser Azure Digital Twins dans cet article, vous devez d’abord *configurer une instance Azure Digital Twins*. Vous devez également disposer des autorisations nécessaires pour l’utiliser. Si vous disposez déjà d’une instance Azure Digital Twins configurée, vous pouvez utiliser cette instance.

Dans le cas contraire, suivez les instructions indiquées dans [Configurer une instance et l’authentification](../articles/digital-twins/how-to-set-up-instance-portal.md). Les instructions contiennent des informations qui vous aideront à vérifier que vous avez correctement effectué chaque étape.

Une fois l’instance configurée, notez les valeurs qui suivent. Vous aurez besoin de ces valeurs pour vous connecter à l’instance :
* Le *nom d’hôte* de l’instance. Vous trouverez le [nom d’hôte dans le portail Azure](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values).
* L’*abonnement Azure* que vous avez utilisé pour créer l’instance. Vous pouvez fournir soit son nom, soit son ID. Vous trouverez l’abonnement dans la page **Vue d’ensemble** de votre instance sur le [portail Azure](https://portal.azure.com).
