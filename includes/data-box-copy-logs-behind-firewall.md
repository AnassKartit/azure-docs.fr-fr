Si les règles de pare-feu sont définies sur le compte de stockage de votre Data Box, vous risquez de ne pas pouvoir accéder aux journaux de copie à partir du portail Azure en utilisant **CHEMIN DU JOURNAL DE COPIE** dans le volet **Vue d’ensemble**. Pour accéder aux journaux, modifiez les paramètres du pare-feu de stockage pour autoriser le système actuel ou utilisez un système qui se trouve dans le réseau du pare-feu.
