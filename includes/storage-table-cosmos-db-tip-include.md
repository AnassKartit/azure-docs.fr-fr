> [!TIP]
> Dans cet article, le contenu s’applique au stockage de Table Azure d’origine. Cependant, il existe désormais une autre offre pour le stockage de tables : l’API Table Azure Cosmos DB. Cette API offre des performances plus élevées et une plus grande disponibilité, une distribution mondiale et des index secondaires automatiques. Elle est également disponible en mode [serverless](../articles/cosmos-db/serverless.md) basé sur la consommation. Il existe des [différences de fonctionnalités](../articles/cosmos-db/table-api-faq.yml) entre l’API Table d’Azure Cosmos DB et celle du Stockage Table Azure. Pour plus d’informations, consultez [API Table Azure Cosmos DB](../articles/cosmos-db/table-introduction.md).
