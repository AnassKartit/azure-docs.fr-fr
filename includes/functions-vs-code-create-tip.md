> [!TIP]
> Par défaut, les ressources Azure nécessaires à votre application de fonction sont créées d’après le nom d’application de fonction que vous indiquez. Par défaut, elles sont également créées dans le même nouveau groupe de ressources avec l’application de fonction. Si vous souhaitez personnaliser les noms de ces ressources ou réutiliser des ressources existantes, vous devez plutôt [publier le projet à l’aide des options de création avancées](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options).
