## <a name="prerequisites"></a>Prérequis

Pour suivre ce didacticiel, vous avez besoin d’un fichier de disque dur virtuel (VHD) chargé sur le compte stockage Azure pour le labo dans lequel vous souhaitez créer l’image personnalisée. Pour charger un fichier VHD sur votre compte de stockage, suivez les instructions de l’un des articles suivants :

- [Chargez un fichier VHD à l’aide de l’utilitaire de ligne de commande AzCopy](../articles/devtest-labs/devtest-lab-upload-vhd-using-azcopy.md).
- [Chargez un fichier VHD à l’aide de Explorateur Stockage Microsoft Azure](../articles/devtest-labs/devtest-lab-upload-vhd-using-storage-explorer.md).
- [Chargez un fichier VHD à l’aide de PowerShell](../articles/devtest-labs/devtest-lab-upload-vhd-using-powershell.md).
