Lorsque vous n’en avez plus besoin, supprimez l’application LUIS. Pour ce faire, sélectionnez **Mes applications** dans le menu en haut à gauche. Sélectionnez les points de suspension (**_..._ *_) à droite du nom de l’application dans la liste des applications, puis _* Supprimer**. Dans la boîte de dialogue contextuelle **Supprimer l’application ?** , sélectionnez **OK**.
