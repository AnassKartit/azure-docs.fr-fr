Cliquez sur **Fichier -> Paramètres de génération**. Cliquez sur **Plateforme Windows universelle**, puis sur **Changer de plateforme**. Si Unity Editor indique que vous devez télécharger d’abord certains composants, téléchargez-les et installez-les. Suivez la capture d’écran ci-dessous pour configurer les paramètres de génération.

:::image type="content" source="./media/object-anchors-quickstarts-unity/aoa-unity-build-settings.png" alt-text="Paramètres de génération":::

Cliquez sur **Générer** et sélectionnez un dossier de sortie. Vous pouvez maintenant générer un projet VS dans le dossier de sortie.
